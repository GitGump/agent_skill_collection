# 公众号热点话题检索策略

## 目录
1. 热点检索的目标
2. 检索方法与技巧
3. 热点筛选与关联
4. 不同赛道的热点特征
5. 实战示例

## 概览
本文档提供有效检索和筛选公众号热点话题的方法，帮助智能体找到与用户主题相关且具有传播潜力的热点内容。

## 核心内容

### 1. 热点检索的目标

#### 核心目标
- 找到与用户主题相关的近期热点
- 识别具有传播潜力的话题
- 确定最佳切入角度
- 确保时效性和相关性

#### 质量标准
- 时效性：近7天内（重要热点）或近30天内（持续性话题）
- 相关性：与用户主题直接相关或可建立逻辑关联
- 传播性：有讨论度、有话题性、能引发共鸣
- 价值性：能为读者提供新信息、新观点或新角度

### 2. 检索方法与技巧

#### 步骤1：确定检索关键词

**基础关键词**：
- 直接使用用户提供的主题
- 提取主题中的核心词汇
- 转换为搜索友好的表达

**扩展关键词**：
- 同义词和近义词
- 相关概念的英文缩写
- 行业术语和流行词
- 关联领域的词汇

**组合策略**：
- `[主题] + 热点`、`[主题] + 趋势`、`[主题] + 新闻`
- `[主题] + 讨论`、`[主题] + 关注`、`[主题] + 曝光`
- `[主题] + 最新`、`[主题] + 突破`、`[主题] + 震撼`

#### 步骤2：执行检索

**搜索来源**：
1. **搜索引擎**（百度、Google）
   - 关键词：`[主题] + 公众号文章`
   - 关注近期高阅读量文章
   - 查看百度热榜、微博热搜

2. **热点聚合平台**：
   - 百度热榜：http://top.baidu.com
   - 微博热搜：https://s.weibo.com/top/summary
   - 今日头条热榜：https://www.toutiao.com/hot-event/hot-board
   - 知乎热榜：https://www.zhihu.com/hot

3. **行业垂直平台**：
   - 科技：36氪、虎嗅、极客公园
   - 财经：华尔街见闻、财新网
   - 教育：芥末堆、多知网
   - 职场：领英、脉脉

4. **社交媒体**：
   - 微信搜一搜：查看"公众号"标签下的热门文章
   - 知乎：查看相关话题的高赞回答
   - 小红书：查看相关笔记的点赞量

#### 步骤3：筛选与评估

**评估维度**：
1. **时效性**：
   - 发布时间是否在近7天/30天内
   - 是否是突发新闻或持续讨论的话题
   - 有无新的进展或后续

2. **相关性**：
   - 直接相关：话题与主题完全一致
   - 间接相关：话题与主题在同一领域
   - 概念关联：可以建立逻辑关联（如：AI→未来工作→职场发展）

3. **传播性**：
   - 阅读量/点赞量/评论量是否高
   - 是否有多家媒体/博主讨论
   - 是否有争议性或讨论空间

4. **价值性**：
   - 是否提供了新信息
   - 是否能引发读者思考
   - 是否有实用价值或情感价值

**筛选标准**：
- 保留时效性强、相关性高、传播性广的话题
- 剔除时效性弱、相关性低、传播性差的话题
- 对于间接相关的话题，评估能否建立合理的关联

### 3. 热点筛选与关联

#### 分类方法

**直接相关热点**：
- 定义：热点话题与用户主题直接重合
- 处理：直接使用，无需额外关联
- 示例：用户主题"AI大模型" → 热点"GPT-4升级"

**间接相关热点**：
- 定义：热点话题与用户主题在同一领域或相关领域
- 处理：找到关联逻辑，作为切入角度
- 示例：用户主题"职场沟通" → 热点"00后整顿职场" → 关联点：新时代的沟通方式

**概念关联热点**：
- 定义：热点话题与用户主题可通过概念建立关联
- 处理：提炼共同点，设计关联逻辑
- 示例：用户主题"情感治愈" → 热点"MBTI爆火" → 关联点：通过自我认知实现治愈

#### 关联策略

**策略1：因果关联**
- 逻辑：热点事件 → 导致/影响了 → 主题相关内容
- 示例：用户主题"家庭教育" → 热点"双减政策" → 关联：双减政策如何影响家庭教育方式

**策略2：对比关联**
- 逻辑：热点中的现象 vs 主题相关的现象
- 示例：用户主题"职场效率" → 热点"远程办公兴起" → 关联：远程办公与传统办公的效率对比

**策略3：趋势关联**
- 逻辑：热点是某个大趋势的体现 → 主题是这个趋势的组成部分
- 示例：用户主题"数字营销" → 热点"AI生成内容爆火" → 关联：AI时代下的数字营销新趋势

**策略4：问题关联**
- 逻辑：热点反映的问题 → 主题提供解决方案
- 示例：用户主题"时间管理" → 热点"年轻人内卷焦虑" → 关联：如何通过时间管理缓解焦虑

#### 热点排序

当找到多个相关热点时，按以下优先级排序：

1. **直接相关 > 间接相关 > 概念关联**
2. **时效性强 > 时效性弱**
3. **传播度高 > 传播度低**
4. **价值性强 > 价值性弱**

### 4. 不同赛道的热点特征

#### 科技赛道
**热点特征**：
- 技术突破、产品发布、重大合作
- 政策变化、行业报告、融资新闻
- 时效性强，更新快

**检索关键词**：
- `AI突破`、`大模型发布`、`芯片技术`、`5G应用`
- `融资消息`、`IPO动态`、`财报发布`

**典型热点**：
- GPT-4升级、国产大模型发布、AI绘画工具爆火
- 芯片制裁、自动驾驶进展、元宇宙降温

#### 情感赛道
**热点特征**：
- 社会现象、情感话题、人际关系
- 心理学概念、自我成长、治愈话题
- 周期性热点（节日、季节、星座）

**检索关键词**：
- `情感话题`、`治愈系`、`人际关系`、`自我成长`
- `MBTI`、`焦虑`、`孤独`、`边界感`

**典型热点**：
- MBTI人格测试爆火、00后整顿职场、边界感讨论
- 独处的能力、拒绝内卷、躺平vs奋斗

#### 职场赛道
**热点特征**：
- 就业形势、职场现象、管理方式
- 技能需求、薪资变化、行业趋势
- 年轻人职场观变化

**检索关键词**：
- `职场新规`、`裁员潮`、`远程办公`、`职场沟通`
- `00后职场`、`35岁危机`、`副业`

**典型热点**：
- 00后整顿职场、35岁危机、远程办公常态化
- 裁员潮、AI取代工作、职场PUA

#### 教育赛道
**热点特征**：
- 政策变化、教育改革、考试动态
- 素质教育、家庭教育、心理健康
- 科技与教育结合

**检索关键词**：
- `双减`、`中考改革`、`高考政策`、`家庭教育`
- `AI教育`、`心理健康`、`课外辅导`

**典型热点**：
- 双减政策落地、中考改革方案、AI教育应用
- 心理健康问题、家庭教育焦虑、校外培训整顿

#### 商业/财经赛道
**热点特征**：
- 市场动态、公司新闻、经济政策
- 行业趋势、投资机会、消费变化
- 国际经济形势

**检索关键词**：
- `股市动态`、`经济政策`、`消费趋势`、`行业发展`
- `独角兽企业`、`IPO`、`并购`

**典型热点**：
- 新能源汽车价格战、大厂裁员、消费降级
- 房地产市场、股市波动、人民币汇率

### 5. 实战示例

#### 示例1：科技类热点检索

**用户主题**：AI大模型

**检索过程**：

1. **确定关键词**：
   - 基础：AI大模型、人工智能、GPT
   - 扩展：ChatGPT、文心一言、通义千问、Llama
   - 组合：`AI大模型 + 热点`、`GPT + 最新`、`人工智能 + 突破`

2. **执行检索**：
   - 搜索`AI大模型 最新突破` → 发现"GPT-4升级"新闻
   - 搜索`国产大模型发布` → 发现"文心4.0发布"
   - 搜索微博热搜 → 发现"AI绘画工具Midjourney V6发布"

3. **筛选评估**：

| 热点 | 时效性 | 相关性 | 传播性 | 价值性 | 综合评分 |
|------|--------|--------|--------|--------|----------|
| GPT-4升级 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 优秀 |
| 文心4.0发布 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 良好 |
| Midjourney V6 | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | 一般 |

4. **选择热点**：
   - 主热点：GPT-4升级（时效性强、相关性高、传播广）
   - 辅助热点：文心4.0发布（国内视角）

**文章切入点**：
- 标题：《大模型战争下半场：GPT-4升级vs国产崛起，谁能笑到最后？》
- 角度：通过GPT-4升级这一热点，引出大模型竞争格局，分析国产大模型的机遇和挑战

## 约束与注意事项
- 热点检索要确保时效性，优先选择近期的热点
- 关联热点时要逻辑自洽，避免强行关联
- 不要为了蹭热点而偏离主题，热点应服务于主题
- 敏感话题（政治、争议事件）要谨慎使用，避免风险
- 热点不是必须的，如果找不到合适热点，可以不使用
- 保持客观中立，避免对热点事件进行过度评价或误导
