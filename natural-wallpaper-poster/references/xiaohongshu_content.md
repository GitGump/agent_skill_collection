- 图片质量：高清无压缩
- 水印位置：右下角或左下角

- 总字数：100-200字
- 段落简洁，不超过3行
- 使用emoji增加视觉节奏
- 情感共鸣 > 信息堆砌

## 自动化文案生成函数

当需要根据壁纸内容自动生成文案时，使用以下结构：

```python
def generate_xhs_content(wallpaper_theme: str, category: str) -> dict:
    """
    根据壁纸主题生成小红书文案

    Args:
        wallpaper_theme: 壁纸的具体主题描述
        category: 自然元素类别

    Returns:
        包含title, body, tags的字典
    """
    # 情感共鸣句
    opening_templates = [
        "今天，想带你逃进一片{}",
        "给你看一个{}的地方",
        "这张壁纸，想送给此刻需要治愈的你",
        "如果累了，就来看看这片{}",
    ]

    # 画面描述
    description = f"画面是{wallpaper_theme}，"

    # 情感升华
    emotion_templates = [
        "愿这份宁静，能抚平你今天的疲惫。",
        "世界很大，但此刻，这一片风景只属于你。",
        "慢下来，感受一下这个世界的美。",
    ]

    # 话题标签
    tags = [
        "山野收容所",
        f"自然壁纸",
        "治愈系",
        f"{category}壁纸",
        "手机壁纸",
        "每日壁纸",
        "高清壁纸",
    ]

    # 组装文案
    title = opening_templates[0].format(wallpaper_theme[:4])
    body = f"{title}\n\n{description}\n\n{emotion_templates[0]}\n\n📱 壁纸尺寸已适配iPhone\n💾 长按保存使用\n\n❤️ 喜欢就收藏吧\n\n#{' #'.join(tags)}"

    return {
        "title": title,
        "body": body,
        "tags": [f"#{t}" for t in tags]
    }
```
