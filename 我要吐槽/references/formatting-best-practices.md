# 公众号排版最佳实践 —— 我要吐槽

## 概述

本文档提供「我要吐槽」公众号文章的排版规范，确保文章在微信公众号编辑器中呈现最佳效果。

---

## 一、整体基调

### 1.1 分析型文章排版

**核心原则**：简洁、现代、专业感

- 主色调：深灰/黑色（#1a1a1a 或 #2d2d2d）
- 背景色：纯白或微灰（#ffffff 或 #fafafa）
- 字体：系统默认，无衬线字体优先
- 行间距：适中（1.8-2.0倍）
- 段间距：清晰但不过度（16-20px）

### 1.2 吐槽型文章排版

**核心原则**：轻松、有节奏、不过度设计

- 主色调：同上（保持一致的品牌感）
- 背景色：纯白
- 可以适当使用活泼的引用样式
- 段间距可稍紧凑（14-18px）

---

## 二、字体与段落

### 2.1 正文字体

- 字号：17px（适合手机阅读）
- 颜色：#2d2d2d（深灰，不刺眼）
- 行高：2.0（两倍行高，便于阅读）
- 对齐：两端对齐（text-align: justify）
- 字间距：0.3px（略微增加字间距）

### 2.2 段落间距

- 段落之间：18px 底部间距
- 段落首行不缩进（公众号风格）
- 段落长度：每段3-5句为宜，不超过手机屏幕一屏

### 2.3 标题样式

| 层级 | 字号 | 字重 | 颜色 | 间距 |
|------|------|------|------|------|
| 一级标题 | 22px | bold | #1a1a1a | 上32px 下20px |
| 二级标题 | 19px | bold | #1a1a1a | 上28px 下14px |
| 三级标题 | 17px | bold | #333333 | 上24px 下12px |

---

## 三、结构设计

### 3.1 标题与正文分隔

**分析型文章**：
- 可使用数字序号小标题（一、二、三）
- 小标题简洁有力，不是标题党
- 标题与正文之间留有清晰间距

**吐槽型文章**：
- 可使用有趣的小标题，但不过度标题党
- 小标题数量可适当少，保持轻松节奏

### 3.2 引用样式

用于突出核心观点或金句：

```
样式特征：
- 背景色：#f8f9fa（分析型）或 #fafafa（吐槽型）
- 左边框：3px solid #5a5a5a
- 内边距：16px 20px
- 字体颜色：#555555
- 字号：16px
- 字体风格：normal（不斜体）
```

### 3.3 禁止使用的元素

- ❌ 任何形式的分隔线（--- / ———）
- ❌ 过多的小标题（碎片化阅读）
- ❌ 醒目的强调色（避免突兀）
- ❌ 居中对齐的段落（仅标题可居中）
- ❌ 过粗的边框或阴影

---

## 四、配图排版

### 4.1 封面图

- 位置：文章标题下方
- 尺寸：900×383像素（严格）
- 对齐：居中
- 边距：上下各20px
- 圆角：2px（微圆角，现代感）

### 4.2 正文配图

- 尺寸：900×500像素（16:9比例）
- 位置：对应段落后
- 对齐：居中
- 边距：上下各24px
- 图片说明：可选，小号灰色字体

### 4.3 图片处理原则

- 不使用过度饱和的图片
- 不使用带水印的图片
- 确保图片清晰不模糊
- 图片风格与文章气质一致

---

## 五、引用与强调

### 5.1 引用块使用场景

- 核心观点或论断
- 金句或关键总结
- 需要读者特别注意的内容

### 5.2 加粗使用原则

- 用于强调关键词或概念
- 不滥用，每段不超过1-2处加粗
- 避免连续加粗（视觉疲劳）

### 5.3 颜色使用

- 正文只用深灰（#2d2d2d）
- 不使用彩色文字（避免花哨）
- 引用文字可用中灰（#555555）

---

## 六、结尾设计

### 6.1 结尾互动语

文章结尾加一段轻柔的引导语：

**分析型**：
```
你有没有观察过这个现象？欢迎分享你的看法。
```

**吐槽型**：
```
你怎么看这个现象？来评论区聊聊。
```

### 6.2 留白

- 结尾到页面底端至少留48px空白
- 给读者呼吸感，不拥挤

## 七、HTML样式代码

以下是generate_html.py中的核心样式：

```css
/* 正文段落 */
p {
    font-size: 17px;
    line-height: 2.0;
    color: #2d2d2d;
    margin-bottom: 18px;
    text-align: justify;
    letter-spacing: 0.3px;
}

/* 一级标题 */
h1 {
    font-size: 22px;
    font-weight: bold;
    color: #1a1a1a;
    margin-top: 32px;
    margin-bottom: 20px;
}

/* 引用块 */
blockquote {
    background-color: #f8f9fa;
    padding: 16px 20px;
    margin: 20px 0;
    border-left: 3px solid #5a5a5a;
    font-style: normal;
    color: #555555;
    font-size: 16px;
}

/* 图片 */
img {
    max-width: 100%;
    height: auto;
    display: block;
    margin: 24px auto;
}
```

## 九、总结

好的排版标准：
1. **不抢内容的戏**：排版服务于内容，不是主角
2. **清晰易读**：让读者专注于观点，而非视觉干扰
3. **克制有品**：现代简约，不堆砌设计元素
4. **呼吸感充足**：留白恰当，不拥挤
5. **风格统一**：与「我要吐槽」的品牌气质一致
