# 选题策略与标题指南 —— 我要吐槽

## 概述

本文档提供「我要吐槽」公众号的选题框架和标题策略，帮助智能体选择合适的话题并生成有吸引力的标题。

---

## 一、选题范围

### 1.1 核心选题领域

**职场现象**
- 办公室文化（表演性忙碌、会议文化、KPI焦虑）
- 职场关系（上下级沟通、同事边界、996争议）
- 工作与生活（内卷、躺平、副业焦虑）
- 职业变迁（转行焦虑、35岁危机、AI替代）

**消费与生活方式**
- 消费主义（剁手文化、精致穷、智商税）
- 健康焦虑（养生朋克、体检焦虑、保健品）
- 居住焦虑（租房困境、房价、蜗居）
- 出行方式（共享单车、电动车、地铁通勤）

**社交与心理**
- 社交焦虑（社恐、无效社交、朋友圈表演）
- 情感关系（相亲文化、婚恋焦虑、恐婚）
- 代际差异（父母催婚、婆媳关系、育儿焦虑）
- 身份焦虑（容貌焦虑、学历焦虑、阶层焦虑）

**技术与生活**
- 数字生活（手机依赖、短视频成瘾、算法推荐）
- 信息过载（知识付费、碎片化阅读、注意力经济）
- 隐私焦虑（数据泄露、监控焦虑、人脸识别）
- AI冲击（ChatGPT焦虑、自动化恐惧）

**城市生活**
- 通勤困境（通勤时间、交通拥堵、地铁拥挤）
- 城市焦虑（漂泊感、归属感缺失、户口焦虑）
- 城乡差异（小镇青年、返乡潮、县域经济）
- 城市病症（孤独、疏离、时间焦虑）

### 1.2 选题判断标准

一个话题是否适合「我要吐槽」，需满足以下条件：

| 标准 | 说明 | 示例 |
|------|------|------|
| **普遍性** | 能引起广泛共鸣，而非小众现象 | ✅ 通勤焦虑 ❌ 某特定职业的困扰 |
| **深度可挖** | 表象下有机制、心理、文化层面的原因 | ✅ 内卷现象 ❌ 某明星八卦 |
| **时效性** | 与当前社会语境相关 | ✅ AI焦虑 ❌ 过时话题 |
| **讨论空间** | 不是非黑即白的简单问题 | ✅ 副业焦虑 ❌ 简单的道德评判 |
| **可读性** | 能用生动语言呈现，不枯燥 | ✅ 表演性忙碌 ❌ 纯学术话题 |

## 二、创作基调判断

### 2.1 适合严肃分析的话题

以下话题更适合理性、冷静的分析基调：
- 社会结构性问题（内卷、阶层流动、教育焦虑）
- 技术冲击类话题（AI替代、算法推荐、隐私问题）
- 制度性话题（社保、医保、户籍制度）
- 历史/文化变迁类话题（代际差异、城乡变迁）

### 2.2 适合风趣吐槽的话题

以下话题更适合轻松、幽默的吐槽基调：
- 生活趣事类话题（养生朋克、剁手文化、朋友圈表演）
- 职场怪象类话题（表演性忙碌、会议文化、PPT内卷）
- 消费怪象类话题（智商税产品、网红经济、盲盒热）
- 社交怪象类话题（社恐、无效社交、相亲文化）

### 2.3 混合型话题

以下话题可以结合吐槽和分析：
- 工作生活平衡类（先吐槽996荒诞，再分析结构性原因）
- 消费主义类（先调侃消费怪象，再分析心理机制）
- 技术焦虑类（先吐槽算法荒诞，再分析技术逻辑）

## 三、标题策略

### 3.1 标题风格分类

**提问式标题**
引发好奇，适合分析型文章
- 《为什么年轻人不爱接电话》
- 《为什么我们越来越害怕无聊》
- 《算法是如何「驯服」我们的》

**判断式标题**
有观点，适合有明确立场的分析
- 《算法正在重塑我们的审美判断》
- 《副业焦虑是一场集体性的自我剥削》
- 《表演性忙碌是职场的生存策略，不是问题本身》

**现象式标题**
点明现象，适合吐槽型文章
- 《办公室里的表演性忙碌》
- 《当代人的养生焦虑与朋克养生》
- 《朋友圈的精致生活与真实疲惫》

**对比式标题**
制造张力，适合有矛盾点的话题
- 《焦虑的中产与躺平的年轻人》
- 《精致的生活与真实的疲惫》
- 《我们要的自由与我们要的安全感》

### 3.2 标题生成原则

**✅ 好标题的特征**
- 简洁有力（通常不超过15个字）
- 有观点或问题意识
- 不用感叹号
- 不用标题党词汇（震惊、真相、绝了）
- 能引发好奇但不误导

**❌ 避免的标题类型**
- 感叹号标题：《年轻人不爱接电话了！》
- 标题党：《震惊！年轻人竟然不爱接电话了》
- 过度悬念：《关于接电话，你必须知道的一件事》
- 鸡汤标题：《不爱接电话的背后，是成长的代价》
- 过度概括：《现代人的通讯困境》（太抽象）

### 3.3 标题生成流程

1. **提炼核心观点**：文章最核心的论点是什么？
2. **判断基调**：适合提问式、判断式还是现象式？
3. **生成3-5个备选**：尝试不同风格
4. **筛选最简洁有力的**：通常选最朴素但最有力度的

**示例流程**
```
文章核心：年轻人不爱接电话，是因为电话往往意味着工作侵扰，这反映了对时间主权的争夺。

备选标题：
1. 为什么年轻人不爱接电话？（提问式）
2. 不爱接电话的背后是对时间主权的争夺（判断式，略长）
3. 当电话成为一种打扰（现象式）
4. 电话、工作与年轻人的时间焦虑（概念堆砌，弱）

最终选择：《为什么年轻人不爱接电话》或《当电话成为一种打扰》
```

## 五、选题挖掘方法

### 5.1 从日常生活观察

- 注意自己和身边人的「不适应感」或「困惑感」
- 关注社交媒体上的高频吐槽或共鸣话题
- 思考「为什么大家都在...」

### 5.2 从数据与新闻切入

- 关注社会调查数据（如「年轻人XX调查报告」）
- 注意热门新闻事件背后的社会心理
- 思考「这个数据说明了什么」

### 5.3 从对比中发现

- 城乡对比、代际对比、性别对比
- 过去与现在的对比
- 不同群体对同一现象的不同反应

## 六、注意事项

1. **不追逐热点但蹭热度**：热点话题要有深度可挖，不为了流量强行写作
2. **不贩卖焦虑**：分析焦虑的成因，而非放大焦虑本身
3. **不简单批判**：指出问题但要分析原因，避免情绪化攻击
4. **不鸡汤说教**：不提供简单的人生建议，而是帮助读者理解现象
5. **承认复杂性**：好现象不是非黑即白，分析应呈现多面性
