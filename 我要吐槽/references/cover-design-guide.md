# 封面设计指南 —— 我要吐槽

## 概述

本文档提供「我要吐槽」公众号文章封面的设计原则和风格指引，帮助智能体生成符合调性的封面图。

---

## 一、封面图定位

### 1.1 核心原则

封面图应与文章气质相符：
- **分析型文章**：简洁现代，有设计感，不过度吸睛
- **吐槽型文章**：轻松有趣，有梗但不low

### 1.2 避免的倾向

- ❌ 商业广告感（过度精致、产品感强）
- ❌ 猎奇博眼球（夸张视觉、争议画面）
- ❌ 廉价感（低质素材、粗糙拼接）
- ❌ 与文章气质不符（严肃话题配搞笑图）

---

## 二、风格分类

### 2.1 分析型封面

**特征**
- 抽象概念可视化（几何图形、符号、数据图）
- 色彩克制（深灰、深蓝、低饱和度配色）
- 构图简洁，留白充足
- 专业感、现代感

**适用话题**
- 社会结构性问题
- 技术冲击类话题
- 制度性话题
- 历史/文化变迁类话题

**示例风格描述**
```
主题：算法推荐对审美的影响
风格：抽象的算法网络图形，节点和连线构成人脸轮廓
配色：深灰背景，白色线条，点缀淡蓝色
氛围：冷静、理性、有科技感
```

### 2.2 吐槽型封面

**特征**
- 手绘插画风或趣味设计
- 色彩可以活泼但不刺眼
- 有梗但不高俗
- 轻松但不幼稚

**适用话题**
- 生活趣事类话题
- 职场怪象类话题
- 消费怪象类话题
- 社交怪象类话题

**示例风格描述**
```
主题：办公室里的表演性忙碌
风格：手绘插画，一个打字员夸张地敲键盘，屏幕上显示的是淘宝
配色：淡黄背景，黑色线条，点缀红色（表现急迫感）
氛围：轻松、有梗、不刻薄
```

### 2.3 混合型封面

**特征**
- 结合两种风格元素
- 有设计感但不过度严肃
- 有趣味但不轻浮

**适用话题**
- 工作生活平衡类
- 消费主义类
- 技术焦虑类（如AI）

## 三、技术规格

### 3.1 尺寸要求

- **严格尺寸**：900×383 像素（2.35:1比例）
- 微信公众号封面标准尺寸，不可偏离

### 3.2 构图原则

- **视觉焦点居中或略偏左**：公众号列表页右侧可能被裁切
- **重要元素避开边缘**：至少留20px边距
- **不依赖文字**：封面不带标题文字

### 3.3 文件格式

- **首选格式**：PNG（无损，支持透明）
- **备选格式**：JPG（照片类）

## 五、生成提示词模板

### 5.1 分析型封面提示词

```
主题：{文章核心概念}
风格：简洁现代的抽象设计
元素：
  - 抽象图形：{具体描述，如几何图形、网络节点、数据可视化}
  - 配色：{配色方案}
  - 构图：简洁居中，留白充足
尺寸：900×383像素
氛围：理性、专业、不猎奇
```

**示例**
```
主题：算法推荐对审美的规训
风格：简洁现代的抽象设计
元素：
  - 抽象图形：网络节点构成的人脸轮廓，节点间有连线
  - 配色：深灰背景(#2d2d2d)，白色线条，淡蓝节点(#5a8faf)
  - 构图：人脸轮廓居中，周围留白
尺寸：900×383像素
氛围：冷静、有科技感、引发思考
```

### 5.2 吐槽型封面提示词

```
主题：{文章吐槽现象}
风格：轻松的手绘插画风
元素：
  - 主体：{具体场景描述}
  - 配色：{配色方案}
  - 氛围：有趣但不低俗，有梗但不刻薄
尺寸：900×383像素
特点：夸张但不过度，幽默但保持克制
```

**示例**
```
主题：办公室里的表演性忙碌
风格：轻松的手绘插画风
元素：
  - 主体：一个打字员夸张地敲击键盘，眉头紧锁，但屏幕显示的是购物网站
  - 配色：淡黄背景(#fff8dc)，黑色线条，点缀橙红色表现急迫感(#ff6b42)
  - 氛围：有梗、不刻薄、引发共鸣
尺寸：900×383像素
特点：夸张表现「忙碌」的动作，但不丑化人物
```

## 六、常见问题

### 6.1 封面与文章气质不符怎么办？

重新审视文章基调，调整封面风格：
- 文章偏分析但封面太活泼 → 调整为简洁抽象风
- 文章偏吐槽但封面太严肃 → 增加趣味元素

### 6.2 抽象概念如何可视化？

| 抽象概念 | 可视化方式 |
|---------|-----------|
| 时间焦虑 | 时钟、沙漏、速度线 |
| 信息过载 | 碎片化文字、网络节点 |
| 边界模糊 | 渗透的色块、交界线 |
| 算法控制 | 网络、节点、数据流 |
| 消费欲望 | 购物车符号、箭头指向 |
| 社交表演 | 面具、滤镜、舞台光 |

### 6.3 如何避免封面同质化？

- 即使同类话题，也尝试不同的视觉隐喻
- 定期更新配色方案
- 同一主题多次生成时，尝试不同构图

## 七、质量检查清单

生成封面后，检查以下要点：

- [ ] 尺寸是否为900×383像素？
- [ ] 风格是否与文章基调相符？
- [ ] 配色是否克制不刺眼？
- [ ] 构图是否简洁不杂乱？
- [ ] 重要元素是否在安全区域内？
- [ ] 是否避免了文字？（封面不带标题）
- [ ] 是否没有猎奇或争议元素？

---

## 八、总结

好的封面图标准：
1. **符合文章气质**
2. **不猎奇博眼球**
3. **有设计感但不堆砌**
4. **引发好奇但不误导**
5. **技术规格严格达标**
