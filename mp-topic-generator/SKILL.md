# 公众号选题生成器 (mp-topic-generator)

## 技能目标

根据用户输入的主题或领域，系统化生成 5-10 个公众号文章选题方向，分类输出，帮助内容创作者快速完成选题策划。

## 触发条件

以下情况应启用本技能：
- 用户输入"帮我生成选题"、"给我几个选题方向"、"公众号写什么"等
- 用户提供一个主题/领域，需要输出多个角度的创作方向
- 用户希望对某个话题进行公众号内容策划

## 选题分类体系

参考 `references/topic-categories.md` 了解各分类的详细定义、适用场景和写作技巧。

核心分类（优先覆盖以下4-6类）：

| 分类名称 | 核心定位 | 推荐频率 |
|---------|---------|---------|
| 爆款吸睛 | 反常识、高争议、强情绪，追求传播 | ★★★★★ |
| 干货方法 | 实用工具、步骤拆解、可执行清单 | ★★★★☆ |
| 人物故事 | 真实人物经历、命运转折、情感共鸣 | ★★★★☆ |
| 痛点共鸣 | 击中情绪痛点、说出读者心声 | ★★★★★ |
| 深度观点 | 独特视角、反直觉洞察、思想深度 | ★★★☆☆ |
| 趋势洞察 | 行业动态、时代变化、前沿预判 | ★★★☆☆ |
| 实战案例 | 真实场景还原、经验总结、避坑指南 | ★★★★☆ |
| 情感叙事 | 温情故事、生活感悟、人性共鸣 | ★★★☆☆ |

## 输出执行流程

### 第一步：理解主题

收到用户主题后，在心中明确：
1. 这个主题的核心受众是谁？（年龄、职业、关注点）
2. 受众的典型痛点和期待是什么？
3. 该主题在公众号语境下最常见的切入点有哪些？

### 第二步：选择分类组合

根据主题性质，从分类体系中选取 4-6 个最适合的分类，确保：
- 必须包含"爆款吸睛"或"痛点共鸣"至少一个
- 必须包含"干货方法"或"实战案例"至少一个
- 每个主题至少生成 5 个、最多 10 个选题
- 同一分类下可生成 1-2 个选题

### 第三步：生成选题详情

每个选题必须包含以下四个字段：

1. **文章标题**：吸引点击的标题，可使用数字、悬念、对比、痛点等技巧
2. **核心思想**：用一句话概括文章的核心观点或价值主张（25字以内）
3. **创作风格**：建议的写作风格（如：冷幽默、深度叙事、清单体、对话体、故事体等）
4. **推荐指数**：⭐1-5星，综合考虑话题热度、传播潜力、创作难度

### 第四步：输出格式规范

严格按照以下 Markdown 格式输出，分类清晰，便于阅读：

```
## 📊 选题总览
- 主题：{用户输入的主题}
- 生成选题数：{N} 个
- 覆盖分类：{分类1}、{分类2}...

---

- **核心思想**：{一句话概括}
- **创作风格**：{风格描述}
- **推荐指数**：⭐⭐⭐⭐⭐

---

- **创作风格**：{风格描述}
- **推荐指数**：⭐⭐⭐⭐

---

```

## 选题质量标准

生成每个标题前，对照以下标准自查：

- [ ] 标题是否能引发「我想看看」的冲动？
- [ ] 核心思想是否有独特视角，而非泛泛而谈？
- [ ] 创作风格是否与该分类匹配？
- [ ] 推荐指数是否客观评估了传播潜力？

## 标题写作技巧参考

详见 `references/topic-categories.md` 中的「标题公式库」章节。

常用公式速查：
- **数字体**：「7个信号说明你……」「做到这3点，……」
- **反转体**：「所有人都说……，但其实……」「我以为……，结果……」
- **痛点体**：「为什么你……总是失败」「那些……的人，后来怎样了」
- **悬念体**：「他用了……，彻底改变了……」「这件事，没人告诉你」
- **对比体**：「同样……，差距为什么这么大」「……的人和……的人，10年后」

## 注意事项

- 标题不使用标点符号结尾（句号除外），避免口号化
- 避免过度夸张导致标题党反感
- 每个分类的选题要有差异化，不重复同一角度
- 推荐指数 5星 极少给，只给真正有爆款潜质的
- 创作风格描述要具体（如「第一人称叙事+冷幽默结尾」优于「幽默风格」）
