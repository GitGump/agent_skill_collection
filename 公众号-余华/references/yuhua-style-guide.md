## 概览
本文档详细阐述余华的写作风格，帮助智能体准确把握其语言特色、结构特点和表达方式，确保生成的文案符合其独特的风格。

## 核心内容

### 1. 风格概述

**余华的核心风格定位**：
- **苦难叙事者**：关注普通人的苦难命运，写尽人间疾苦
- **极简主义者**：用最少的文字表达最深的情感
- **冷峻旁观者**：用冷静的笔触写残酷的现实，不煽情但深情
- **时间记录者**：对时间和记忆有着独特的敏感

**风格关键词**：
- 极简
- 克制
- 冷峻
- 苦难
- 荒诞
- 象征
- 留白

### 2. 语言特点

#### 2.1 极简克制

**特点**：
- 用最少的文字表达最深的情感
- 不说透，但让人懂
- 克制比煽情更有力量

**示例**：
```
书面语：人生的苦难往往让人感到无比的悲伤和痛苦。
余华风格：人活着，就是受苦。苦着苦着，就习惯了。
```

#### 2.2 冷峻疏离

**特点**：
- 用冷静的笔触写残酷的现实
- 不煽情，但情感在字里行间
- 像旁观者，但比谁都清楚

**示例**：
```
煽情表达：这是一个多么悲惨的故事啊，让人泪流满面。
余华风格：他死了。死在冬天，雪还没化。
```

#### 2.3 白描手法

**特点**：
- 不用形容词，只用动词和名词
- 画面感强，像电影镜头
- 越简单，越有力

**示例**：
```
华丽表达：夕阳西下，金色的余晖洒满了整个村庄，显得格外美丽和宁静。
余华风格：太阳落山了。村子黑了。
```

#### 2.4 重复与变奏

**特点**：
- 通过重复强化主题
- 每次重复都有细微变化
- 像音乐中的变奏曲

**示例**：
```
第一次：他走了。
第二次：他真的走了。
第三次：他走了，再也没回来。
```

### 3. 结构特点

#### 3.1 开篇：平静如水

**特点**：
- 从一个平静的场景开始
- 不张扬，不渲染
- 平静中暗藏张力

**示例结构**：
```
场景切入 → 人物出场 → 暗示命运

示例：
那天早上，雾很大。
他站在门口，抽了根烟。
烟抽完了，他就走了。
```

#### 3.2 中段：暗流涌动

**特点**：
- 表面平静，内里汹涌
- 命运在不知不觉中改变
- 苦难一点点积累

**示例结构**：
```
平静叙述 → 事件叠加 → 命运转折

示例：
他开始干活。
干了一天，又一天。
干了一年，又一年。
后来，他干不动了。
干不动的人，就老了。
```

#### 3.3 结尾：余韵悠长

**特点**：
- 不说完，留给人想象
- 余音绕梁，三日不绝
- 读完之后，久久回味

**示例结构（余韵悠长）**：
```
事件结束 → 留下空白 → 意味深长

示例：
他死了。
死的时候，身边没有人。
没有人知道他什么时候死的。
也许，他自己也不知道。
```

### 4. 修辞手法

#### 4.1 象征

**特点**：
- 用具体事物象征抽象概念
- 一物多义，耐人寻味
- 越简单的事物，象征越深刻

**示例**：
```
象征：那棵树站了很多年。站着站着，就老了。
象征意义：时间的流逝，生命的无常。
```

#### 4.2 隐喻

**特点**：
- 不说破，但让人懂
- 隐喻比明喻更有力量
- 读者自己领悟

**示例**：
```
隐喻：他像一条河，流了很多年，最后流进海里。
真实含义：人的一生，最终归于虚无。
```

#### 4.3 荒诞

**特点**：
- 用荒诞写真实
- 越荒诞，越真实
- 黑色幽默，笑中带泪

**示例**：
```
荒诞：他死了，葬礼上却放起了喜庆的音乐。
效果：荒诞中见真实，讽刺现实。
```

#### 4.4 留白

**特点**：
- 不说透，留空白
- 空白处，情感最浓
- 让读者自己填补

**示例**：
```
留白：她看着他，没说话。
他没说话，也没看她。
两个人就这样站着。
（空白处：情感的暗流）
```

#### 4.5 时间感

**特点**：
- 对时间极度敏感
- 时间在不知不觉中流逝
- 时间是最好的叙事者

**示例**：
```
时间：那是很多年前的事了。
很多年前，是多久？
久到他已经忘了。
忘了，就是真的忘了。
```

### 5. 标题风格

#### 5.1 标题特点

**余华风格的标题特点**：
- 简洁，不花哨
- 像陈述，不像标题
- 有力量，不说教
- 看着普通，细想有深意

**示例**：
```
普通标题：关于人生苦难的思考
余华风格：
活着
许三观卖血记
在细雨中呼喊
第七天
```

#### 5.2 常用标题模板

**模板1：一个动作**
```
结构：[动词] + [对象]
示例：
- 活着
- 等待
- 离去
- 归来
```

**模板2：一个场景**
```
结构：[地点/时间] + [事件]
示例：
- 细雨中
- 黄昏里
- 冬天
- 那年夏天
```

**模板3：一个人**
```
结构：[人物] + [特征/动作]
示例：
- 卖血的人
- 等待的人
- 沉默的父亲
- 远行的儿子
```

**模板4：时间与记忆**
```
结构：[时间] + [记忆/遗忘]
示例：
- 往事
- 遗忘
- 时间的灰烬
- 记忆的碎片
```

### 6. 段落特点

#### 6.1 段落长度

**特点**：
- 短段落为主
- 一句一段，节奏感强
- 短句是艺术，不是简单

**示例**：
```
✅ 正确（短）：
他走了。
走得很远。
远到看不见了。
看不见，就是真的走了。

✅ 也可以（稍长，但有节奏）：
他走了，走得很远。
远到看不见了。
看不见，就是真的走了。
走了的人，不会再回来。
```

#### 6.2 段落间距

**特点**：
- 段落之间空一行
- 留白是呼吸，是节奏
- 不要堆得太满

**示例**：
```
第一段

第二段

第三段
```

### 7. 开头和结尾

#### 7.1 开头设计

**特点**：
- 从一个场景开始
- 不端架子，不装高深
- 平静中暗藏张力

**示例**：
```
那天，雨下得很大。
他在屋檐下站着，站了很久。
久到雨停了，他还在站着。
站着站着，天就黑了。
```

#### 7.2 结尾设计

**特点**：
- 不说完，留给人想象
- 或者戛然而止，余韵悠长
- 读完之后，久久回味

**示例（戛然而止）**：
```
他走了。
再也没有回来。
```

**示例（余韵悠长）**：
```
很多年后，他已经忘了那件事。
忘了，就是真的忘了。
但有时候，他会突然想起来。
想起来，又忘了。
就这样，过了一辈子。
```

### 8. 实战示例

#### 示例1：人生主题文章

**标题**：活着

**开头**：
他活着。
活着就是活着，没什么特别的。
但他活着，就已经很不容易了。
不容易的事，往往没什么人知道。

**正文片段**：
他每天干活。
干完活，就睡觉。
睡醒了，再干活。
就这样，过了一天又一天。
一天一天地过，就是一辈子。
一辈子有多长？
长到他已经忘了什么时候开始。
也忘了什么时候结束。
结束是什么时候？
结束就是结束，没什么特别的。

**结尾**：
后来他死了。
死的时候，身边没有人。
没有人知道他什么时候死的。
也许，他自己也不知道。
活着的时候，他不知道活着是什么。
死了，也不知道死是什么。
这就是活着。

#### 示例2：职场主题文章

**标题**：等待

**开头**：
他在等。
等什么？不知道。
不知道等什么，就是等待。
等待的人，往往不知道自己在等什么。

**正文片段**：
他等了很多年。
等升职，等加薪，等机会。
等来等去，什么也没等到。
没等到，就是等待的结果。
结果是什么？
结果就是，他还在等。
等一个不知道会不会来的东西。
来了，是运气。
不来，是命。

**结尾**：
后来他不再等了。
不等了，就是不等了。
不等了，反而等到了。
等到了什么？
等到了不等。

#### 示例3：情感主题文章

**标题**：遗忘

**开头**：
他忘了。
忘了什么？忘了一个人。
一个人，说忘就忘了。
忘了，就是真的忘了。

**正文片段**：
他努力想记起。
想记起那个人的样子。
样子是模糊的。
模糊的东西，往往是记不起来的。
记不起来，就是遗忘。
遗忘是什么？
遗忘就是，你以为你记得，其实你已经忘了。

**结尾**：
后来他又想起来了。
想起来，又忘了。
忘了，又想起来。
就这样，过了一辈子。
一辈子，就是记住和遗忘。
记住的，会遗忘。
遗忘的，会记住。

### 9. 注意事项

#### 9.1 避免的事项

**避免**：
- 过于华丽的辞藻
- 过于煽情或说教
- 过于复杂（让人看不懂）
- 过于冗长（拖泥带水）
- 过于直白（一竿子捅到底）
- 过于聪明（显得油滑）

#### 9.2 必须做到的事项

**必须做到**：
- 极简，用最少的文字表达最深的情感
- 克制，不煽情但深情
- 冷峻，用冷静的笔触写残酷的现实
- 留白，不说透，留给人想象
- 短句，节奏感强
- 象征，用具体事物表达抽象概念

#### 9.3 风格一致性

**保持一致**：
- 整篇文章风格统一
- 不要一会儿余华，一会儿刘震云
- 不要一会儿极简，一会儿华丽
- 让读者感觉是同一个人在说话
- 同一个人说话，但不是同一个人
- 是谁？是那个讲故事的人

## 约束与注意事项
- 文章创作必须严格遵循余华的写作风格
- 保持极简、克制、冷峻的特点
- 关注普通人的苦难命运
- 不说透，留余味
- 避免华丽辞藻和过于说教
- 保持风格一致性，不要混合多种风格
