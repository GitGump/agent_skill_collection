## 概览
本文档提供微信公众号文章的专业排版优化技巧，帮助智能体生成适合手机端阅读、视觉舒适、易于传播的排版内容。

## 核心内容

### 1. 排版的核心原则

#### 1.1 可读性第一
- **字体大小**：正文16号，小标题18号，大标题24号
- **行间距**：1.75-2.0倍（推荐1.75）
- **段落间距**：段落之间空一行
- **段落长度**：每段3-5句话，最多不超过5行（手机屏幕）

#### 1.2 视觉舒适
- **配色统一**：主色+辅助色不超过3种
- **留白充足**：不要塞满屏幕，给眼睛休息空间
- **对比适度**：重点突出，但不要过度强调

#### 1.3 引导阅读
- **清晰结构**：用小标题分割内容
- **重点突出**：关键信息用加粗或引用
- **节奏变化**：交替使用文字、图片、列表

#### 1.4 互动引导
- **结尾引导**：点赞、在看、转发
- **提问设计**：引导读者留言
- **话题设置**：使用热门标签

### 2. 结构化排版

#### 2.1 标题层级

**大标题（主标题）**：
- 样式：加粗、24号字
- 要求：吸引眼球、包含关键词
- 示例：`# 大模型战争下半场：谁能笑到最后？`

**小标题（章节标题）**：
- 样式：加粗、18号字
- 数量：3-5个
- 要求：概括章节内容、简洁明了
- 示例：`## 01 技术突破：GPT-4的升级`

**子小标题（更细的分割）**：
- 样式：加粗、16号字或直接加粗文字
- 使用场景：章节内再细分
- 示例：`### 核心能力提升` 或直接 `**核心能力提升**`

#### 2.2 开头设计

**摘要/引语**：
- 位置：大标题下方、正文前
- 样式：引用样式、16号字、灰色或浅色
- 内容：概括文章核心观点或制造悬念
- 示例：
  ```
  > OpenAI刚刚宣布了GPT-4的重大升级，性能提升30%。
  > 这对整个AI行业意味着什么？
  ```

**注意：不使用分隔线**

#### 2.3 段落分割

**段落长度控制**：
- 每段3-5句话
- 手机屏幕显示不超过5行
- 长段落拆分为2-3个短段

**段落间距**：
- 段落之间空一行
- 不同章节之间空两行

**示例对比**：

❌ **错误**：段落过长
```
OpenAI刚刚宣布了GPT-4的重大升级，性能提升30%，这在整个AI行业引起了巨大反响。这次升级主要包括了三个方面的改进：首先是模型能力的全面提升，能够处理更复杂的任务；其次是响应速度的优化，让用户获得更好的体验；最后是安全性的增强，减少了错误和偏见。这些改进对于企业和个人用户来说都意味着新的机遇，但同时也带来了挑战，特别是对于竞争对手来说，压力更大了。
```

✅ **正确**：段落适中
```
OpenAI刚刚宣布了GPT-4的重大升级，性能提升30%，这在整个AI行业引起了巨大反响。

这次升级主要包括了三个方面的改进：

首先是模型能力的全面提升，能够处理更复杂的任务；
其次是响应速度的优化，让用户获得更好的体验；
最后是安全性的增强，减少了错误和偏见。

这些改进对于企业和个人用户来说都意味着新的机遇，但同时也带来了挑战。
```

### 3. 视觉优化技巧

#### 3.1 重点突出

**加粗使用**：
- 用于：关键词、核心观点、重要数据
- 原则：不要过度使用，每段不超过2处
- 示例：`**GPT-4的升级**标志着AI进入新阶段`

**引用样式**：
- 用于：金句、重要观点、数据引用
- 样式：灰色背景、缩进、不同字体
- 示例：
  ```
  > "真正厉害的人，都懂得边界感。"
  >  知乎高赞回答
  ```

**特殊标记**：
- 用于：数字、标签、强调
- 示例：
  - 数字：`>100万`用户、`3个`方法
  - 标签：`#职场沟通`、`#AI大模型`
  - 强调：🚀重要！、💡小贴士

#### 3.2 列表化要点

**有序列表**：
- 用于：步骤、排名、序列
- 示例：
  ```
  1. 第一步：确定主题
  2. 第二步：检索热点
  3. 第三步：创作内容
  ```

**无序列表**：
- 用于：并列要点、注意事项
- 示例：
  ```
  - 字体大小：16号
  - 行间距：1.75
  - 段落间距：空一行
  ```

**图标列表**：
- 用于：提升视觉效果、增加趣味性
- 示例：
  ```
  ✓ 已完成
  ✗ 待完成
  ⚠ 注意
  💡 建议
  📊 数据
  🎯 目标
  ```

#### 3.3 配色建议

**科技赛道**：
- 主色：蓝色系（科技感）
- 辅色：灰色、白色
- 强调色：橙色或绿色（用于重点）

**情感赛道**：
- 主色：粉色、浅紫色（温暖）
- 辅色：米色、浅灰
- 强调色：深蓝或深红（用于金句）

**职场赛道**：
- 主色：深蓝、黑色（专业）
- 辅色：灰色、白色
- 强调色：橙色或红色（用于数据）

**教育赛道**：
- 主色：绿色、蓝色（成长）
- 辅色：浅黄、米色
- 强调色：红色（用于重点）

### 4. 配图插入策略

#### 4.1 配图位置

**封面图**：
- 位置：大标题下方、摘要下方
- 尺寸：900×383像素（2.35:1比例）
- 要求：简洁大气、文字清晰
- **Markdown图片标签**：`![封面图](./images/封面图.png)` 或 `![封面图](https://图片URL)`

**正文配图**：
- 位置1：每个小标题后（推荐）
- 位置2：重要段落后（金句、数据）
- 位置3：文章结尾前（引导阅读）
- 尺寸：900×500像素（16:9比例）
- 要求：与内容相关、风格统一
- **Markdown图片标签**：`![配图1](./images/配图1.png)` 或 `![配图1](https://图片URL)`

**图片标签规范**：
- 必须为所有图片添加Markdown图片标签
- 标签格式：`![图片描述](图片路径)`
- 本地图片路径：`./images/图片名.png`
- URL图片路径：`https://example.com/image.png`
- 确保所有图片都有对应的标签，这样生成的HTML才能完整显示

**分隔图**：
- 位置：章节之间（可选）
- 尺寸：900×200像素（装饰性）
- 要求：简洁、不干扰阅读

#### 4.2 配图标注

在Markdown中标注配图插入位置：

```
[配图1：GPT-4技术架构图]
📌 插入位置：小标题"01 技术突破"之后

[配图2：AI应用场景示意]
📌 插入位置：第4段之后

[配图3：未来趋势预测图]
📌 插入位置：结尾总结之前
```

#### 4.3 配图数量建议

- **短文**（1000字以内）：1-2张
- **中等长度**（1500字左右）：3-4张
- **长文**（2000字以上）：5-6张

原则：不要过多（影响阅读速度），也不要过少（视觉单调）

**点赞引导**：
```

请点个**"赞"**👍

```

### 6. 不同赛道的排版特点

#### 科技赛道

**特点**：专业、简洁、数据化

**排版重点**：
- 使用更多图表和数据可视化
- 小标题用编号（01、02、03）
- 配图偏科技感（蓝色系、图表）
- 金句突出，用引用样式

**示例结构**：
```
# 标题

> 摘要：数据或新闻



## 01 技术突破

[正文+图表]

> "核心观点"

## 02 应用场景

[正文+案例]



点赞+在看+转发👇
```

#### 情感赛道

**特点**：温暖、细腻、治愈

**排版重点**：
- 使用柔和配色（粉色、浅紫）
- 段落偏短，节奏缓慢
- 配图偏插画、风景（温暖风格）
- 金句频繁，用引用样式

**示例结构**：
```
# 标题

> 摘要：情感共鸣



## 01

[小故事+情感描述]

> "治愈系金句"

## 02

[观点+建议]



欢迎在评论区分享你的故事~
点赞和"在看"，温暖你我❤️
```

#### 职场赛道

**特点**：实用、清晰、可操作

**排版重点**：
- 使用列表化要点
- 步骤用编号（1、2、3）
- 配图偏职场场景
- 重要方法加粗突出

**示例结构**：
```
# 标题

> 摘要：痛点引入



## 方法一：XXX

1. 第一步
2. 第二步
3. 第三步

> "核心建议"

## 方法二：XXX

[正文+案例]



点赞+在看，一起进步💪
```

#### 教育赛道

**特点**：专业、易懂、有温度

**排版重点**：
- 平衡专业性和可读性
- 使用图标和列表
- 配图偏教育场景（儿童、学习）
- 重点内容用框或底色突出

**示例结构**：
```
# 标题

> 摘要：教育现象引入



## 01 现状分析

[正文+数据图表]

💡 **小贴士**

## 02 解决方案

1. 方法A
2. 方法B
3. 方法C



转发给需要的家长~
点赞+在看，一起成长📚
```

### 7. 排版检查清单

发布前检查以下事项：

#### 结构检查
- [ ] 大标题是否加粗突出
- [ ] 小标题数量是否合适（3-5个）
- [ ] 段落长度是否适中（3-5句话）
- [ ] 段落间距是否充足（空一行）
- [ ] 是否有明显的章节分割

#### 视觉检查
- [ ] 重点内容是否加粗（不要过度）
- [ ] 金句是否用引用样式
- [ ] 配图位置是否合理
- [ ] 配图尺寸是否合适
- [ ] 配图风格是否统一

#### 互动检查
- [ ] 结尾是否有引导互动
- [ ] 是否设置了提问
- [ ] 是否有话题标签

#### 细节检查
- [ ] 字体大小是否合适（正文16号）
- [ ] 行间距是否合适（1.75）
- [ ] 是否有错别字
- [ ] 标点符号是否规范
- [ ] 链接是否有效

#### 手机端检查
- [ ] 预览手机端显示效果
- [ ] 图片是否加载正常
- [ ] 长段落是否需要拆分
- [ ] 重点是否突出可见
- [ ] 整体视觉是否舒适

## 排版优化示例

### 示例1：优化前（原始Markdown）

```
# 如何提高工作效率

提高工作效率是每个职场人都关心的话题。有几种方法可以尝试：制定清晰的目标是第一步，只有目标明确才能有方向感；其次是合理安排时间，避免拖延；最后是保持专注，减少干扰。

除了这些基本方法，还有一些技巧可以提升效率。比如使用番茄工作法，每25分钟专注工作，然后休息5分钟；还有重要紧急四象限法，按照重要性和紧急程度安排任务；还有就是学会拒绝，不要承担太多无关紧要的事情。

如果你能坚持下去，一定会看到效果。
```

### 示例2：优化后（专业排版）

```
# 如何提高工作效率

> 提高工作效率是每个职场人都关心的话题。
> 掌握正确的方法，让你事半功倍！



## 01 制定清晰目标

只有目标明确，才能有方向感。

- ✅ 短期目标：本周/本月要完成什么
- ✅ 长期目标：季度/年度要达成什么
- ✅ 可量化：目标要具体，不要模糊

> "没有目标的人，永远在为有目标的人打工。"

## 02 合理安排时间

避免拖延，提高执行力。

1. **番茄工作法**：每25分钟专注工作，休息5分钟
2. **重要紧急四象限**：按重要性和紧急程度安排任务
3. **二八定律**：20%的重要工作产生80%的价值

## 03 保持专注

减少干扰，提升效率。

- 关闭不必要的通知
- 使用专注模式App
- 设定"深度工作"时间段

💡 **小贴士**：专注是一种能力，需要刻意练习。



坚持下去，你一定能看到效果！💪

**点赞 + "在看" + 转发**
是对我最好的支持！👇
```

## 约束与注意事项
- 排版要服务于内容，不要为了排版而排版
- 不同的赛道有不同的排版风格，要灵活调整
- 手机端体验是第一位的，优先考虑手机阅读
- 不要过度使用特效和装饰，保持简洁
- 配图要适量，不要过多影响阅读速度
- 定期更新排版风格，跟随公众号生态变化
