# 排版指南

## 排版原则

**像一份杂志，不是一份报告。**

电影杂志的排版特点：
- 信息密度高但不拥挤
- 有呼吸感的留白
- 视觉元素帮助导航
- 重点内容被「提亮」

---

## 标题层级

### 主标题

```
# 标题（居中，不带书名号）
```

**要求**：
- 16-20号字
- 可以加粗
- 可以用深色（如 #1a1a1a）
- 禁止：感叹号、流量词

### 副标题（可选）

```
## 副标题
```

**用于**：
- 解释主标题
- 点明角度
- 制造悬念

### 小标题（章节导航）

```
## 小标题
```

**用于**：
- 主要章节
- 重点段落
- 内容切换

### 无小标题情况

如果文章较短（2000字内），可以不用小标题，用段落间距区分内容层次。

---

## 内容区块

### 普通段落

```
这是普通段落内容。段落之间空一行表示分隔。
```

**格式**：
- 首行不缩进
- 段间距大于行间距
- 字号：16px
- 行高：1.8
- 字间距：正常

### 引用（台词/原文）

```
> 「台词内容或原文引用」
> —— 《片名》第XX集/XX分钟
```

**格式**：
- 左边框线（深色）
- 斜体或浅色字
- 上下留白
- 标注出处

### 重点强调

**使用加粗**，不要用斜体：
```
这是一个**需要强调**的内容。
```

**不要使用**：
- 全部大写
- 颜色文字（除非是装饰）
- 下划线（除非是链接）

## 列表使用

### 场景推荐列表

```
🎬 如果你喜欢这部剧，这些也不该错过：
- 《推荐1》—— 一句话理由
- 《推荐2》—— 一句话理由

🎭 如果你喜欢这个演员，这些是必看：
- 《推荐3》—— 一句话理由
- 《推荐4》—— 一句话理由
```

### 分层推荐

```
🎧 入门级（如果你从没听过他们）
- 「歌曲名」—— 推荐理由

🔥 死忠级（如果你已经爱了，这些是必听）
- 「歌曲名」—— 推荐理由
```

## 图片排版

### 封面图

```
![封面描述](图片URL)

正文从这里开始...
```

- 封面图宽度 = 文章内容宽度
- 图片与正文间距：24px以上

### 正文配图

```
...段落内容...

![配图描述](图片URL)

...继续段落...
```

- 图片居中
- 图片下方可加简短图注（可选）
- 图片与正文间距：16px

### 图注格式（可选）

```
*图注内容*
```

---

## 间距规范

| 元素 | 间距 |
|------|------|
| 标题与正文 | 24-32px |
| 小标题与正文 | 16-24px |
| 段落与段落 | 16px |
| 引用块上下 | 24px |
| 图片与文字 | 16-24px |
| 列表项之间 | 8-12px |

## 字体设置

### Markdown 原生

- 标题：系统默认衬线/无衬线
- 正文：系统默认

### HTML 输出

```css
body {
  font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif;
  font-size: 16px;
  line-height: 1.8;
  color: #333;
}

h1 {
  font-size: 24px;
  font-weight: bold;
  margin-bottom: 24px;
}

h2 {
  font-size: 20px;
  font-weight: bold;
  margin-top: 32px;
  margin-bottom: 16px;
}

blockquote {
  border-left: 3px solid #666;
  padding-left: 16px;
  color: #666;
  font-style: italic;
  margin: 24px 0;
}
```

## 颜色方案

### 推荐配色

| 用途 | 颜色 | 色值 |
|------|------|------|
| 正文 | 深灰 | #333 / #444 |
| 标题 | 黑色 | #1a1a1a |
| 辅助文字 | 浅灰 | #666 / #888 |
| 引用边框 | 深色 | #333 / #666 |
| 强调色 | 品牌色 | #E74C3C 等 |

### 避免

- 全篇白色背景 + 黑色文字（太刺眼）
- 五颜六色混搭
- 荧光色

## 结尾互动

文章结尾可以加一句轻柔的引导，但不强制：

```
---

*你在评论区聊聊你印象最深的那个镜头。*
```

或：

```
---

*你最喜欢哪个角色？*
```

---

## 常见排版错误

| 错误 | 修正 |
|------|------|
| 标题用 emoji 开头 | emoji放标题后面或不用 |
| 大量 `---` 分隔线 | 用段落间距代替 |
| 列表项过长 | 拆分成多行或精简 |
| 全篇加粗/斜体 | 只强调真正重要的 |
| 引用块里套引用块 | 只用一层引用 |
