# 封面设计指南

## 封面设计原则

封面是文章的第一帧画面。它应该：
1. 传递影片气质
2. 有视觉冲击力
3. 让人想点进来

**三不要**：
- 不要文字（文字放文章里）
- 不要剧照拼贴（太像海报）
- 不要纯色（太敷衍）

---

## 风格定位

### 根据影片类型

| 类型 | 配色 | 氛围 | 示例元素 |
|------|------|------|---------|
| 犯罪/悬疑 | 冷灰/深蓝/暗红 | 紧张/压抑 | 剪影、烟雾、雨 |
| 爱情/浪漫 | 暖橙/粉/米白 | 温柔/怀旧 | 花瓣、光斑、窗 |
| 科幻/奇幻 | 蓝紫/霓虹/冷白 | 未来/神秘 | 城市天际线、星空 |
| 喜剧/轻松 | 明黄/浅绿/淡粉 | 轻快/活泼 | 几何图形、抽象 |
| 恐怖/惊悚 | 黑/血红/暗紫 | 恐惧/不安 | 阴影、镜子、门 |
| 历史/战争 | 土黄/橄榄/铁灰 | 沉重/沧桑 | 旧照片质感、纹理 |

### 根据导演风格

| 导演 | 风格参考 |
|------|---------|
| 王家卫 | 霓虹、雨夜、慢动作、暧昧 |
| 昆汀·塔伦蒂诺 | 血浆、波普、色块、暴力美学 |
| 李安 | 留白、水墨、东方意境 |
| 诺兰 | 冷峻、结构感、时间元素 |
| 是枝裕和 | 日常、阳光、家庭场景 |

## 构图建议

### 常用构图方式

| 构图 | 适用 | 说明 |
|------|------|------|
| 中心构图 | 人物特写 | 主体在正中，聚焦感强 |
| 三分法 | 场景/关系 | 主体在交叉点 |
| 对角线 | 动态/张力 | 引导视线，增加动感 |
| 框架构图 | 人物/情绪 | 门/窗/镜子框住主体 |
| 留白 | 艺术感/诗意 | 大量空白，突出主体 |

### 氛围营造

- **光影**：侧光=戏剧感；逆光=诗意；底光=诡异
- **虚实**：前景虚化=层次；主体清晰=聚焦
- **色调**：冷调=疏离；暖调=亲近

## AI绘图提示词模板

### 基础模板

```
[影片类型] movie poster style, [核心意象], [色调], [光影], cinematic lighting, [构图], no text, [分辨率]
```

### 示例

**犯罪剧封面**：
```
Crime thriller movie poster style, a lone figure silhouette standing in rain at night, cold blue and deep red tones, dramatic side lighting, cinematic, center composition, no text, 4K, hyperrealistic
```

**爱情剧封面**：
```
Romance drama movie poster style, two hands almost touching through soft light, warm orange and cream tones, gentle backlighting, intimate close-up composition, no text, 4K, dreamy atmosphere
```

**王家卫风格**：
```
Wong Kar-wai style, neon reflections on wet street, lone woman in silhouette, magenta and cyan tones, cinematic moody lighting, wide angle, no text, 4K, film grain texture
```

## 封面尺寸规格

- **宽高比**：2.35:1（电影比例）
- **像素尺寸**：900x383 像素
- **文件格式**：PNG/JPG
- **分辨率**：72 DPI（屏幕显示）

---

## 避免的设计

1. ❌ 纯色背景+字体（那是PPT）
2. ❌ 直接用剧照（版权问题+无设计感）
3. ❌ 多个演员脸拼在一起（那是演员表）
4. ❌ 动画风格（除非原片是动画）
5. ❌ 过多元素叠加（视觉混乱）
