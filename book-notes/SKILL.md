# 读书笔记技能 (book-notes)

## 技能概述

本技能用于对输入的书籍进行深度拆解与解读，生成结构化的读书笔记。支持多种笔记模式（简洁版、标准版、详细版），以及从不同身份视角（职场人/创业者、学生/学习者、家长/教育者等）输出个性化的读书感想。

## 输入处理

### 获取书籍信息

当用户提供书名时，执行以下步骤获取书籍信息：

1. **使用网络搜索** 获取书籍的基本信息：
   - 作者、出版社、出版年份
   - 书籍简介与核心主题
   - 主要章节结构
   - 业界评价与影响力

2. **整理书籍信息**，形成初步理解：
   - 确定书籍类型（商业管理、自我成长、小说文学、历史社科等）
   - 识别核心受众群体
   - 提炼书籍的核心价值主张

### 信息不足时的处理

如果搜索结果有限或用户有特定书籍内容需要深入分析，应：

1. 询问用户是否可以提供书籍的具体内容、摘要或重点章节
2. 询问用户对书籍的已有了解程度
3. 基于用户提供的信息进行深入分析

## 笔记模块设计

### 支持的笔记模式

| 模式 | 适用场景 | 输出内容 |
|------|----------|----------|
| **简洁版** | 快速浏览、备忘 | 3-5个核心启发 + 简短总结 |
| **标准版** | 日常阅读记录 | 书籍简介 + 核心要点 + 精彩摘录 + 个人感想 |
| **详细版** | 深度学习、系统整理 | 书籍简介 + 章节拆解 + 核心观点 + 深度分析 + 思考与应用 |

### 各模块内容要求

#### 1. 书籍基本信息
- 书名、作者、出版社、出版年份
- 一句话推荐语（20字以内）
- 适合阅读人群

#### 2. 核心要点（3-5个）
每个要点包含：
- **观点名称**：简洁有力的标题
- **原文摘录**：书中原话（加引号标注）
- **深度解读**：用自己的话解释观点含义
- **实践指引**：如何在生活中应用

#### 3. 精彩摘录
选取3-10个金句：
- 原文内容
- 摘录原因（为什么这句话打动你）
- 个人感悟

#### 4. 章节拆解（详细版）
按逻辑顺序梳理各章节：
- 章节主题
- 核心内容
- 与其他章节的关联
- 个人理解

#### 5. 深度分析与思考
- 书籍的局限性或争议点
- 与同类书籍的比较
- 延伸阅读建议

## 身份视角读书感想

### 支持的身份视角

#### 职场人/创业者视角
- 关注点：职场技能提升、领导力、商业洞察、创业经验
- 分析维度：可复用的方法论、可落地的行动方案、职业发展启发

#### 学生/学习者视角
- 关注点：学习方法、知识体系构建、考试/升学相关
- 分析维度：知识点的理解与应用、学以致用的建议

#### 家长/教育者视角
- 关注点：教育理念、亲子沟通、家庭教育方法
- 分析维度：如何传递给孩子、适合的家庭场景应用

### 输出格式

```
📖 【XX视角下的读书感想】

🎯 我最受益的三个观点：
1. ...
2. ...
3. ...

💼 【如果是职场人视角】实际应用场景：
   - 场景1：...
   - 场景2：...

🌟 行动建议：
   - 立即可做：...
   - 本周计划：...
   - 长期坚持：...
```

## 特殊要求处理

当用户有特殊要求时，灵活调整输出：

| 特殊要求 | 处理方式 |
|----------|----------|
| 指定章节 | 重点拆解指定章节，补充必要的上下文 |
| 特定问题 | 围绕问题深入分析，提供针对性解答 |
| 对比阅读 | 提供与其他书籍的对比分析 |
| 特定视角 | 深入该视角，提供定制化感想 |

## 输出格式规范

### 整体结构

```
# 📚 《书名》读书笔记

> 作者 | 出版社 | 出版年份
> ⭐ 推荐指数：XX/5

## 一、书籍简介
...

## 二、核心要点
### 1. 第一个核心观点
### 2. 第二个核心观点
...

## 三、精彩摘录
...

## 四、章节拆解（详细模式）
...

## 五、XX视角下的读书感想
...

## 六、思考与应用
...

## 七、延伸阅读推荐
...
```

### 格式要求

- 使用 Markdown 格式输出
- 适度使用 emoji 增强可读性
- 重要观点加粗标注
- 摘录使用引用格式
- 结构层次分明，便于后续回顾

## 工作流程

1. **接收输入**：获取书名或用户指定的书籍内容
2. **信息获取**：通过网络搜索获取书籍基本信息
3. **确认需求**：询问用户需要的笔记模式和身份视角（可跳过使用默认）
4. **生成笔记**：按照选定模式生成完整读书笔记
5. **输出结果**：以 Markdown 格式呈现

## 默认设置

- 默认笔记模式：标准版
- 默认身份视角：职场人/创业者视角
- 默认语言：简体中文

## 交互提示

在生成笔记时，可使用以下引导语：

- "正在为您搜索《书名》的相关信息..."
- "根据您的选择，我将生成【详细版】读书笔记，包含完整的章节拆解..."
- "以下是职场人视角下的读书感想，聚焦于实际工作中的应用..."

## 注意事项

1. **信息准确性**：搜索结果可能存在偏差，对于关键信息应注明来源
2. **个人化**：在模板基础上加入个人理解和独特见解
3. **实用性**：确保感想部分可落地、可执行
4. **适度原则**：不追求面面俱到，突出最打动你的内容
