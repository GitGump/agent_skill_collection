# 书籍分类与推荐场景

## 商业管理类

### 适合职场人/创业者视角的书籍特征
- 实战方法论
- 案例丰富
- 可落地的工具/框架
- 行业洞察

### 推荐场景
- 团队管理：《卓有成效的管理者》《领导力必读12篇》
- 战略思维：《从优秀到卓越》《竞争战略》
- 运营效率：《精益创业》《高效能人士的七个习惯》
- 营销推广：《定位》《营销战》

## 自我成长类

### 适合所有视角的书籍特征
- 通用性强
- 心理洞察深刻
- 实操性强

### 推荐场景
- 习惯养成：《原子习惯》《习惯的力量》
- 思维提升：《思考，快与慢》《批判性思维》
- 情绪管理：《情绪的力量》《情商》
- 时间管理：《番茄工作法》《要事第一》

## 小说文学类

### 不同视角的关注点
- **职场人**：人物塑造、情节设计、写作技巧
- **学生**：文学价值、写作手法、人物分析
- **家长**：教育意义、价值观传递、亲子共读

### 推荐场景
- 经典名著：《百年孤独》《了不起的盖茨比》
- 现代文学：《活着》《围城》
- 科幻小说：《三体》《沙丘》

## 历史社科类

### 推荐场景
- 历史传记：《曾国藩传》《苏东坡传》
- 社会洞察：《乌合之众》《乡土中国》
- 经济思维：《国富论》《资本论》

## 思维方法论类

### 推荐场景
- 逻辑思维：《金字塔原理》《批判性思维工具》
- 系统思考：《系统之美》《第五项修炼》
- 创新思维：《创新者的窘境》《水平思考法》

## 不同视角的读书重点

### 职场人/创业者

```
核心问题：这本书能给我带来什么实际价值？

关注维度：
1. 方法论是否可复制？
2. 案例是否有借鉴意义？
3. 如何落地执行？
4. 对我的业务/职业有什么启发？
```

### 学生/学习者

```
核心问题：这本书能帮我解决什么学习问题？

关注维度：
1. 有哪些高效学习方法？
2. 如何构建知识体系？
3. 有哪些考试/升学相关的启发？
4. 如何学以致用？
```

### 家长/教育者

```
核心问题：这本书适合推荐给孩子吗？有什么教育启示？

关注维度：
1. 适合什么年龄段的孩子？
2. 有什么正面的价值观？
3. 如何与孩子共读/讨论？
4. 有哪些家庭教育的方法可以借鉴？
```
