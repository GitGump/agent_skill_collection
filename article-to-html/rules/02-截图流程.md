# 截图流程

## 切片先行设计的优势

由于 HTML 中每个 `.slice-card` 固定 1080×1440px，总高度 = N × 1440px（精确整数倍），**无需精确测量内容高度**，截图和切片流程大幅简化。

## 方案 A：Puppeteer 精确截图（推荐）

> 用 puppeteer + 系统 Chrome 截图，精确控制视口大小，无需安装额外浏览器。

```bash
# 在 HTML 文件所在目录运行
node ~/.workbuddy/skills/article-to-html/scripts/screenshot.js <html文件> [输出目录]
```

脚本会自动：
1. 启动临时 HTTP 服务
2. 测量 `.container` 的实际高度（应等于 N × 1440）
3. 设置精确视口并截图完整长图
4. 调用 slice-image.js 生成小红书切片图

## 方案 B：HTTP 服务 + Chrome DevTools Protocol

> **不要用 `file://` 协议**，会被浏览器安全策略拦截。必须通过 HTTP 服务打开。

```
□ 0. 启动临时 HTTP 服务（如果没有在跑的）
   在 HTML 所在目录启动：
   python3 -m http.server 8899 --directory <html所在目录> &

   然后用 http://localhost:8899/文件名.html 打开

□ 1. 打开页面
   mcp: new_page → url: http://localhost:8899/xxx.html

□ 2. 清除背景和边距
   mcp: evaluate_script →
   () => {
     document.body.style.cssText = 'background:transparent;padding:0;margin:0;display:block;min-height:auto';
     const c = document.querySelector('.container');
     if (c) c.style.margin = '0';
     return 'done';
   }

□ 3. 计算视口大小（简化：切片数量 × 1440）
   mcp: evaluate_script →
   () => {
     const cards = document.querySelectorAll('.slice-card');
     const totalHeight = cards.length * 1440;
     return { width: 1080, height: totalHeight, sliceCount: cards.length };
   }
   → mcp: resize_page(width, height)

□ 4. 截图完整长图（用 PNG 不用 JPG）
   mcp: take_screenshot → fullPage: true, format: "png", filePath: 输出路径_full.png

□ 5. 生成小红书 3:4 切片图
   node ~/.workbuddy/skills/article-to-html/scripts/slice-image.js <输出路径_full.png>
   → 自动生成 xxx_slice_01.png, xxx_slice_02.png, ...（每张 1080×1440）

□ 6. 关闭页面
   mcp: close_page
```

## 方案 C：手动截图

```
1. 浏览器打开 HTML
2. F12 → 设备模拟 → 自定义 1080 × {切片数×1440}
3. Cmd+Shift+P → "Capture full size screenshot"
4. 得到完整长图后，手动用 ImageMagick 切片：
   magick 完整长图.png -crop 1080x1440+0+0 slice_%02d.png
```

## 切片规则

- **切片尺寸**：1080 × 1440（3:4 比例，与小红书推荐规格一致）
- **精确对齐**：由于 HTML 中每个 .slice-card 固定 1440px，切片边界与内容边界完美对齐
- **命名规范**：`{原文件名}_slice_01.png`, `{原文件名}_slice_02.png`, ...
- **切片后输出文件清单**：
  ```
  E:\codefile\trae\mp_article\文生图\
  └── {标题}-{时间戳}-V版本号\
      ├── .meta.txt           ← 元信息（标题/时间戳/版本号）
      ├── index.html          ← 源 HTML
      ├── index_full.png       ← 完整长图（需手动截图保存）
      └── index_slice_01.png  ← 小红书切片图 × N张
  ```

## 常见坑

| 问题 | 原因 | 解法 |
|------|------|------|
| `file://` 被拦截 | 浏览器安全策略 | 用 HTTP 服务（步骤 0） |
| 截图是 JPG 模糊 | 默认格式不对 | 指定 `format: "png"` |
| 字体没加载 | Google Fonts 需要时间 | 截图前等 2 秒 |
| 残留进程报错 | 上次没关干净 | `pkill -f "chrome-devtools-mcp"` |
| 切片内容被截断 | 没用 .slice-card 结构或内容溢出 | 确保每个 .slice-card 内容不超出 1296px 可用高度 |
| 总高度不是 1440 整数倍 | .slice-card 使用了 min-height 或 height:auto | 确保 .slice-card 使用固定 height:1440px |
