# 商机管理核心指标体系

## 一、获客指标

### 1.1 商机总量
- **定义**：统计周期内新增的商机总数
- **计算**：`COUNT(*)`
- **目标**：环比增长 > 10%
- **预警**：环比下降 > 20%

### 1.2 月度新增趋势
- **定义**：按月统计的新增商机数量
- **计算**：`GROUP BY MONTH(申请日期)`
- **目标**：平稳增长或季节性波动在预期内
- **预警**：连续2个月下降

### 1.3 渠道分布
- **定义**：各渠道来源的商机数量及占比
- **计算**：`GROUP BY 客户来源`
- **目标**：前3大渠道占比 < 70%（不过度依赖单一渠道）
- **预警**：单一渠道占比 > 50%

---

## 二、转化指标

### 2.1 试用转化率
- **定义**：客户状态为"试用"的商机占比
- **计算**：`SUM(客户状态='试用') / COUNT(*) * 100%`
- **目标**：> 70%
- **预警**：< 50%

### 2.2 各渠道转化率
- **定义**：按渠道统计的试用转化率
- **计算**：`SUM(客户状态='试用') / COUNT(*) GROUP BY 客户来源`
- **目标**：各渠道 > 60%
- **预警**：任何渠道 < 30%

### 2.3 转化趋势
- **定义**：按月统计的试用转化率变化
- **计算**：`SUM(客户状态='试用') / COUNT(*) GROUP BY MONTH`
- **目标**：保持稳定或上升
- **预警**：连续下降或降幅 > 15%

---

## 三、跟进指标

### 3.1 平均跟进次数
- **定义**：所有商机的人均跟进次数
- **计算**：`AVG(跟进次数)`
- **目标**：> 1.5次
- **预警**：< 0.8次

### 3.2 零跟进率
- **定义**：跟进次数为0的商机占比
- **计算**：`SUM(跟进次数=0) / COUNT(*) * 100%`
- **目标**：< 30%
- **预警**：> 50%

### 3.3 跟进率
- **定义**：跟进次数 > 0的商机占比
- **计算**：`SUM(跟进次数>0) / COUNT(*) * 100%`
- **目标**：> 70%
- **预警**：< 50%

### 3.4 深度跟进率
- **定义**：跟进次数 ≥ 2的商机占比
- **计算**：`SUM(跟进次数>=2) / COUNT(*) * 100%`
- **目标**：> 30%
- **预警**：< 15%

### 3.5 各渠道跟进率
- **定义**：按渠道统计的跟进率
- **计算**：`SUM(跟进次数>0) / COUNT(*) GROUP BY 客户来源`
- **目标**：各渠道 > 60%
- **预警**：任何渠道 < 30%

---

## 四、数据质量指标

### 4.1 手机号完整率
- **定义**：手机号非空的记录占比
- **计算**：`SUM(手机号 IS NOT NULL AND 手机号 != '') / COUNT(*) * 100%`
- **目标**：> 90%
- **预警**：< 70%

### 4.2 需求完整率
- **定义**：需求字段非空的记录占比
- **计算**：`SUM(需求 IS NOT NULL AND 需求 != '') / COUNT(*) * 100%`
- **目标**：> 60%
- **预警**：< 30%

### 4.3 数据质量综合评分
- **定义**：各字段完整率的加权平均
- **计算**：`(手机号完整率*0.4 + 需求完整率*0.3 + 试用结束日期完整率*0.3)`
- **目标**：> 75分
- **预警**：< 50分

## 五、渠道效能矩阵

### 定义
将渠道按"商机量"和"转化率"分为四类：

```
        高转化率
            ↑
            │
    ┌───────┼───────┐
    │  重点 │ 放量  │
    │  维护 │ 渠道  │
    ├───────┼───────┤
    │  淘汰 │ 优化  │
    │  考虑 │ 对象  │
    └───────┼───────┘
            │
        低转化率
            ←─── 低量 ─── 高量 ──→
```

### 分类标准
- **重点维护**（高量+高转化）：量大且转化好，重点维护
- **放量渠道**（低量+高转化）：转化好但量小，加大投入
- **优化对象**（高量+低转化）：量大但转化差，需优化流程
- **淘汰考虑**（低量+低转化）：双低，考虑淘汰

## 六、基准值参考

| 指标 | 优秀 | 良好 | 及格 | 需改善 |
|------|------|------|------|--------|
| 试用转化率 | >85% | 70-85% | 50-70% | <50% |
| 零跟进率 | <20% | 20-35% | 35-50% | >50% |
| 平均跟进次数 | >2.5 | 1.5-2.5 | 0.8-1.5 | <0.8 |
| 深度跟进率 | >40% | 25-40% | 15-25% | <15% |
| 手机号完整率 | >95% | 85-95% | 70-85% | <70% |
| 需求完整率 | >80% | 60-80% | 30-60% | <30% |

---

## 七、分析维度优先级

1. **P0 - 必须分析**：转化率、零跟进率、数据质量
2. **P1 - 重要分析**：渠道效能、跟进分布、月度趋势
3. **P2 - 补充分析**：深度跟进、客户状态明细、特殊案例分析
