# 非虚构写作风格指南 —— 真实故事计划调性

## 概述

本文档是「真实故事计划」公众号写作风格的系统性指南，供智能体在创作非虚构文章时参考遵循。核心理念：**从生命里拿出来的故事**，是纪实的，是人的，是有温度但不煽情的。

---

## 一、核心写作哲学

### 1.1 非虚构 ≠ 新闻报道

| 维度 | 新闻报道 | 真实故事计划非虚构 |
|------|---------|----------------|
| 目标 | 传递信息 | 呈现人的处境与内心 |
| 视角 | 旁观者、客观 | 第一人称、介入的 |
| 语言 | 简洁精准，去情感化 | 有质感，克制而不冷漠 |
| 结构 | 倒金字塔，最重要信息前置 | 慢进入，情节推进，余韵收尾 |
| 结论 | 明确，有判断 | 开放，让读者自己感受 |

### 1.2 「人性的挣扎与微光」是选题核心

不是要写「成功的故事」，不是要写「感动的故事」，而是要写：

- **一个人在某种处境下，如何做出了某个选择**
- **那个选择背后，有什么东西在支撑或撕裂他/她**
- **生活在那之后，有没有变得不同，哪怕只是一点点**

「微光」不是励志，是在困境中仍然存在的、微小的、真实的人情与体面。

### 1.3 不消费苦难

「真实故事计划」最忌讳的写法：

- ❌ 将底层苦难作为展示道具，引发读者的优越感或怜悯
- ❌ 用「感动」作为目的，强行推进情绪
- ❌ 用夸张的悲剧叙述博取流量
- ✅ 将叙述者的处境呈现为：「这就是生活的样子，你也许能在其中认出某种自己」

## 二、叙事视角与人称

### 2.1 第一人称叙述（主力）

绝大多数真实故事计划文章以「我」为核心叙述者：

```
我站在那个砖砌的宿舍门口，已经是下午三点，阳光直射进来，照在我那双还粘着黄泥的球鞋上。
```

- 「我」是经历者，不是全知旁观者
- 「我」有困惑，有局限，看不到全局
- 「我」的理解在故事中不断变化

### 2.2 第一人称叙述 + 还原他人视角

当需要呈现故事中另一个核心人物时，「我」可以描述自己对他/她的观察与猜测，但不能直接宣称进入他人内心：

```
❌ 错误：「他那时候一定很痛苦。」（直接宣称他人内心）
✅ 正确：「他沉默了很长时间，然后把那杯茶端起来，又放下了。」（通过行为呈现）
```

### 2.3 叙述层次

优秀的非虚构叙述常有两条线索：
- **事件线**：发生了什么
- **内心线**：叙述者如何感受、思考、理解

两条线交织，形成叙事的纵深。

## 三、语言风格

### 3.1 具体优于抽象

非虚构写作最忌讳的是**情绪词替代细节**：

```
❌ 弱：「她当时非常难过。」
✅ 强：「她把手机屏幕扣过去，扣在餐桌上，这个动作她做了好几次。」

❌ 弱：「那是一段艰难的时光。」
✅ 强：「那三个月，我每天早上醒来的第一件事就是看账户余额，然后想今天要省掉哪顿饭。」
```

### 3.2 感官细节建立现场

好的非虚构文章，读者应该能「看见」「听见」「闻到」叙述者所处的场景：

- 视觉：光线、色彩、人物的姿势和动作
- 听觉：声音、对话、环境噪音
- 触觉/嗅觉：质地、气味（适度使用）

```
示例：
「工厂里的气味是那种混合了机油、棉絮和汗水的复合气味，我到现在有时候在某个地铁站闻到类似的味道，还会愣一秒。」
```

### 3.3 对话的力量

对话是非虚构写作中最有力的工具，可以：
- 展现人物性格（说话方式就是性格）
- 推进情节
- 传递情感（不用直接说「他很冷漠」，让他的话来说）

**对话写作原则**：
- 对话要真实、口语化，有语气词、停顿、不完整的句子
- 对话后不需要总是加「她说」，可以用动作交代
- 对话与叙述交替，避免大段连续对话

```
示例：
「你会回来吗？」我问。
他没有回答，只是把那个旧行李箱拉到门口，里面发出轻微的滚动声。
```

### 3.4 克制的情绪表达

真实故事计划的情绪是「渗出来的」，不是「喊出来的」：

```
❌ 过度煽情：「泪水夺眶而出，那一刻，我终于明白了生命的意义。」
✅ 克制有力：「我坐在那儿没动，直到服务员来问我要不要续水，我才意识到时间已经过了很久。」
```

## 四、结构设计

### 4.1 开篇：抛入场景，不做自我介绍

**禁止开头**：
```
❌「我叫小李，今年28岁，是一名普通的工厂工人。」
❌「前段时间，一件事让我深深地思考起了人生的意义……」
```

**好的开头**：直接进入一个具体时刻/场景

```
✅「那天他们通知我结果的时候，我正在公司楼下的便利店买一杯热可可。店员问我要不要加料，我站在那里想了很久，最后说不用了。」

✅「师傅第一次带我上桥，是在三月，风很大，站在护栏边上可以看到整条江。」
```

### 4.2 中段：推进，不停顿在情绪里

故事的中段应该像水流——持续流动，不在某个情绪点上停留太久，否则会变成渲染。

**中段节奏控制**：
- 叙事块（3～5 句）→ 一个细节/对话 → 内心独白（1～2 句）→ 继续叙事

**时间线处理**：
- 主要叙述顺序进行
- 可以有回溯（用「那之前」「那之后」引入），但不要太多，避免读者迷失

### 4.3 转折点：全文情感密度最高处

每篇好的非虚构都有一个「转折时刻」——可以是一句话、一个动作、一个选择，在这里故事发生了某种不可逆的改变。

这个时刻需要**放慢节奏、增加细节**：

```
示例：
「手术结束出来，医生找到我，说了什么，我没有听进去多少。我记得的是走廊的光，很亮，白色的，我用手遮了一下眼睛。然后我想到我妈在家等着我的电话，她说过，不管结果怎样，都要第一时间告诉她。我掏出手机，在联系人里找到她，又把手机收回去了。」
```

### 4.4 结尾：留有余地，不作总结

真实故事计划的结尾很少给出人生总结，而是：

- 停在故事现在的时间点上，让读者感受余韵
- 留下一个开放的状态（「现在还没有答案」）
- 用一个细节、一个动作、一句简单的陈述句收束

```
❌ 总结式结尾：「这件事让我明白，人生没有白走的路，每一步都算数。」
✅ 余韵式结尾：「我现在偶尔还会经过那条街。换了两家店，原来的修鞋摊早就不在了。我每次都走得很快，也没什么特别的感觉，就是走过去了。」
```

## 五、选题方向与人物类型

### 5.1 核心选题主题

| 主题 | 人物/场景举例 | 叙事角度 |
|------|------------|---------|
| 职场 | 被裁员的 35 岁中年人、应届生入职头一年 | 个人困境与社会结构的撞击 |
| 情感 | 异地恋结束后、照顾失忆配偶的老人 | 爱的形式与代价 |
| 边缘群体 | 性工作者、流浪汉、精神病患者家属 | 生存边缘的真实处境 |
| 乡土中国 | 留守老人、进城失败后返乡的年轻人 | 城乡之间的撕裂 |
| 城市漂泊 | 北漂/沪漂的普通人、群租房故事 | 异乡人的孤独与坚守 |
| 生死 | 重症病房的故事、殡仪馆工作者 | 死亡面前的人性 |
| 疾病 | 抑郁症患者本人、患者家属 | 疾病如何改变一段关系 |
| 犯罪 | 服刑者家属、受害者家属 | 犯罪阴影下的普通人 |
| 小众职业 | 验尸官、陪诊师、夜班收费员 | 职业本身即故事 |

### 5.2 人物选择原则

优先选择以下类型的主人公：
- **处于生命关键转折点的普通人**（不是名人，不是极端人物）
- **有困境，但不是被困境完全定义的人**（还有自己的欲望、小幽默、惯常的动作）
- **有局限性的人**（犯过错、做出过糟糕的选择）——这让故事更真实

## 六、常见写作错误

### 6.1 符号化人物
```
❌「她是那种典型的坚强女性，一个人扛起了整个家。」
✅ 让她的坚强和脆弱都在具体行为里呈现，读者自己感受
```

### 6.2 滥用比喻

非虚构写作里比喻不宜过多，过度的修辞会破坏真实感：
```
❌「他的眼神里有一片汪洋，盛满了沧桑岁月的重量。」
✅「他看了我一会儿，没有说话，然后把视线转向窗外。」
```

### 6.3 信息过载

真实故事计划不是百科全书，不需要大篇幅背景介绍：
- ❌ 3 段话介绍「外卖行业的现状与数据」
- ✅ 用一个外卖员真实的一天来自然呈现行业状态

### 6.4 结尾升华
```
❌「这段经历让我明白，无论多难，只要心中有光，就能走下去。」
✅ 让故事自然结束，不强行提炼「意义」
```

## 八、AI 辅助创作常见问题清单（必读）

以下是 AI 生成非虚构时的系统性偏差，**每次创作前必须对照自检**，若有任意一项命中，重写该部分。

### 8.1 人物模板化

**症状**：主人公永远是"30岁出头、互联网/大厂、某个中部省份"，背景高度雷同。

**根源**：AI 倾向从"典型案例"数据库抽样，拼贴标准化人物。

**自检问题**：
- 主人公的家乡是哪个具体的县/镇/村？路是什么样的？
- 他/她有什么习惯性动作？（比如"说话前要先清一下嗓子""坐着的时候脚会抖动"）
- 他/她的口头禅是什么？
- 写完人物后，和"30岁互联网大厂被裁"的故事放一起对比——是否除了名字不同，其他完全一样？

**改法**：给人物增加一个具体的、特殊的、不可复制的细节。比如同样是失业的故事，让主人公是一个"在车间里用粉笔画了十二年设备图纸的钳工"，或者"一个在殡仪馆给遗体化妆的入殓师"。**职业越具体，故事越不可替代。**

### 8.2 情节平铺直叙，缺乏戏剧张力

**症状**：故事按时间顺序平铺，"第一步做了什么，第二步做了什么"，读起来像工作报告。

**根源**：AI 擅长总结归纳，不擅长制造"让读者想知道接下来发生了什么"的悬念。

**自检问题**：
- 故事中有没有一个"如果不说出来，读者不可能预料到"的时刻？
- 有没有一个让读者想说"等等，发生了什么"的瞬间？

**改法**：找到故事中真正出人意料的那个节点，用**慢镜头**处理这件事——增加动作细节、环境细节、放慢节奏。而不是用"然后""之后""接着"一笔带过。

### 8.3 对话严重缺席

**症状**：全文几乎没有真实对话，人物靠"他说""我觉得""我想"交代，缺乏质感。

**根源**：AI 生成对话需要更复杂的推理，容易跳过。

**自检问题**：
- 全文对话段落是否少于 5 处？
- 主人公说话时，有没有口语化的停顿、不完整的句子、语气词？

**改法**：
- 每篇至少安排 5～8 段真实对话
- 对话不用"他说"直接交代，可以用动作代替：`他把手里的烟掐灭，说："……"`
- 对话内容要口语化，不完整、有停顿、有方言质感

### 8.4 感官细节缺席

**症状**：场景描写只有"我去了某个地方"，没有光线、声音、气味、温度。

**根源**：AI 默认处理抽象概念，不主动调用感官。

**自检问题**：
- 闭上眼睛读这篇文章，脑海中能形成一个画面吗？
- 有没有一个场景能让人"闻到"或"听到"？

**改法**：
- 每个主要场景至少写一个感官细节（光线/声音/气味/触觉）
- 感官细节不要用形容词堆砌，用具体的名词和动词

### 8.5 内心线断裂

**症状**：只有事件线（发生了什么），没有内心线（我在想什么、感受到了什么、为什么那件事对我重要）。

**根源**：内心描写需要第一人称的主观性和脆弱感，AI 在这方面天然保守。

**自检问题**：
- 叙述者"我"的困惑、犹豫、自我怀疑是否有所呈现？
- 有没有一个内心独白是真实的、哪怕是混乱的、不确定的？

**改法**：
- 在故事关键节点加入内心独白，呈现叙述者当时的真实心理状态
- 内心独白不需要完整，可以是闪念、碎片、不成熟的理解

### 8.6 转场生硬

**症状**：段落之间靠"后来""之后""有一天"硬跳，缺有质感的过渡。

**改法**：
- 用一个具体的场景/物品/感受作为转场
- 不用时间词连接，改用空间/感官/情绪的自然过渡

### 8.7 用词暴露 AI 痕迹

**症状**："组织效能优化"、"结构性调整"、"N+1赔偿"等公文词汇；"以下是他的讲述"格式生硬。

**改法**：
- 人物的语言要口语化、具体、不完美
- "以下是他的讲述"可以自然融入场景，不单独成行
- 避免新闻稿式的裁员描述，改用人物真实的反应：愣住、发抖、不知道该做什么

## 九、快速自检清单（每次创作后使用）

发布/交付前，逐项核对：

- [ ] **人物具体性**：人物有独特职业背景、习惯动作、口头禅，不是模板人
- [ ] **戏剧张力**：至少有一个"意想不到"的时刻，不是流水账
- [ ] **对话密度**：对话 ≥5 处，口语化，有停顿和不完整句
- [ ] **感官细节**：每段主要场景至少一个视觉/听觉/嗅觉/触觉细节
- [ ] **内心线**：有内心独白或心理挣扎，不是纯事件罗列
- [ ] **转场质感**：不用"然后""后来""之后"硬跳
- [ ] **无 AI 痕迹用词**：没有公文词汇、没有格式化的"以下是他的讲述"
- [ ] **结尾有回味**：用具体细节/动作收束，不做总结性陈述
- [ ] **禁止词汇**：无"扎心/破防/泪崩/逆袭/正能量/感动了无数人"

发布前用以下标准检验文章：

- [ ] 开篇是否直接进入具体场景？（无「我叫XXX」式介绍）
- [ ] 是否有足够具体的感官细节？（视觉/听觉/行动细节）
- [ ] 是否有真实的对话或独白？
- [ ] 情绪词是否已被具体行为/细节替代？
- [ ] 主人公是否有立体感？（而非单一标签）
- [ ] 转折时刻是否放慢节奏、深入处理？
- [ ] 结尾是否留有余地？（无说教式总结）
- [ ] 全文是否没有「流量词汇」？（破防/泪崩/扎心/感动了无数人）
- [ ] 是否避免了消费苦难的倾向？
- [ ] 整体字数是否在 2000～4000 字范围内？
