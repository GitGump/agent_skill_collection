# 选题策略、采访框架与标题指南 —— 真实故事计划

## 一、选题策略

### 1.1 什么样的故事值得写？

**核心判断标准（三问法）**：

1. **有人吗？** 这个故事里有一个真实的、有血有肉的人，而不是一种「现象」或「问题」
2. **有困境吗？** 这个人面对了什么无法轻易解决的困境——不是小麻烦，是真正影响人生的困境
3. **有挣扎吗？** 这个人在其中做了选择、付出了代价、或仍在困境里悬而未决

满足这三点，故事就有了基本的非虚构叙事土壤。

### 1.3 选题禁忌

以下类型的故事不符合真实故事计划调性：

- ❌ **单纯的成功故事**：「从农村走出来的孩子，终于在城市站稳脚跟」——若没有困境与代价，就缺乏真实感
- ❌ **极端猎奇故事**：用恐怖、变态、怪异吸引眼球，消费人物的不幸
- ❌ **单纯的受害者叙事**：只呈现被动受苦，没有主体性
- ❌ **明显的正能量框架**：故事一开始就指向「坚强」「奋斗」「感恩」
- ❌ **名人故事**：真实故事计划聚焦普通人

## 二、采访框架（当用户提供素材或模拟采访时）

### 2.1 采访前的核心问题

在开始写作前，智能体应梳理（或向用户追问）以下信息：

**基础信息**（不用写进文章，用来建立人物）：
- 主人公年龄大概多少？
- 在哪里发生的？（城市/农村/工厂/医院……）
- 故事发生在什么时间范围内？

**核心叙事信息**：
- 故事「起点事件」是什么？（是什么让这个故事开始的？）
- 最重要/最难忘的一个场景或时刻是什么？
- 主人公面对的最核心困境是什么？
- 主人公做了哪些选择？付出了什么代价？
- 故事现在在哪里？（已结束/还在进行中）

**细节挖掘**：
- 有没有一段印象深刻的对话？
- 有没有一个很具体的物品/场景？（旧行李箱/一张照片/一个食物……）
- 他/她有什么习惯性的动作或说话方式？

---

### 2.2 素材处理原则

当用户提供采访素材时：

1. **不随意填充**：用户没说到的细节，不要「合理推测」后写进文章，可以以「我不记得清楚了，只是……」等方式处理模糊处
2. **尊重素材原有的情感基调**：如果原始故事是平静叙述的，不要在写作时放大悲剧感
3. **关键时刻放慢**：对于素材中最打动人的那个片段，用更多笔墨展开
4. **剔除「信息型」内容**：用户提供的背景知识（如「外卖行业的运营模式」）尽量通过故事细节呈现，不要直接罗列

## 三、标题指南

### 3.1 真实故事计划标题特征

与一般公众号「爆款标题」不同，真实故事计划的标题风格是：

| 特征 | 说明 |
|-----|-----|
| **朴素** | 不用感叹号，不用煽情词 |
| **具体** | 最好带有职业/地点/人物状态等具体信息 |
| **有重量** | 读起来像一个事实，但含有某种张力 |
| **多义性** | 有时一个标题可以有两层读法 |
| **不解释** | 不在标题里泄露情绪或立场，让读者自己感受 |

### 3.2 代表作品标题分析

```
《赶时间的人》
—— 「赶时间」既是字面意思（外卖骑手的奔跑），也是对现代人处境的隐喻

《长江大桥救下 323 名轻生者》
—— 纯粹的事实陈述，数字让人肃然，没有任何情绪词

《我在奥运会上做黄牛》
—— 第一人称 + 具体场景（奥运会）+ 敏感职业（黄牛），制造反差，引发追问

《被资助长大的失足女孩》
—— 两种身份并置（「被资助」「失足」），产生张力，不做评判
```

### 3.3 标题生成模板（真实故事计划风格）

**类型 A：职业/处境型**
```
结构：[身份/职业] + [状态/动作]
示例：
- 《夜班的收费员》
- 《卖花的女人》
- 《在医院陪诊的人》
```

**类型 B：事件陈述型**
```
结构：[具体事件/数字/场景]
示例：
- 《他在这座城市住了十四年》
- 《我妈把房子卖了》
- 《三次离婚之后》
```

**类型 C：第一人称主观型**
```
结构：「我」 + [做了/经历了/看见了] + [某件具体的事]
示例：
- 《我送走了我爸》
- 《我在精神病院工作了三年》
- 《我决定不再回那个村子》
```

**类型 D：人物处境型**
```
结构：[身份] + [处境/状态]（被动语态制造悬念）
示例：
- 《被退货的新娘》
- 《被留下的孩子》
- 《从工厂消失的父亲》
```

**类型 E：地点/时间锚定型**
```
结构：[地点/时间] + [人的状态]
示例：
- 《在长江边站了十一年》
- 《凌晨四点的厂区》
- 《最后一班车上的人》
```

### 3.4 标题禁忌

- ❌ 感叹号：「太震撼了！」「这个故事让我哭了！」
- ❌ 网络流量词：「扎心」「破防」「泪崩」「感动了无数人」
- ❌ 过度悬念：「没想到……」「结局出乎意料」
- ❌ 定性评价：「最感人」「最伟大」「最坚强」
- ❌ 空洞大词：「生命的意义」「活着的价值」

## 四、公众号发布策略

### 4.1 推送时间
- 非虚构长文适合晚间推送（20:00～22:00）
- 周五、周六晚间读者情绪更容易沉浸

### 4.2 文章开头（摘要）
- 公众号「摘要」字段用 1～2 句话呈现故事核心场景，不是标题的复述
- 示例：「他每天要在长江大桥上来回走十几趟。不是为了锻炼。」

### 4.3 评论区互动引导
- 真实故事计划风格的互动引导：**邀请，不催促**
- 示例：「如果你也有一个想被记录的故事，欢迎在留言区分享，或发邮件给我们。」
- 不使用「点赞+在看+转发」三件套的商业化说法，可以用：「如果这个故事触动了你，点击「在看」，让更多人看见这类普通人的故事。」
