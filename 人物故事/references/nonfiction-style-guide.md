# 非虚构人物报道写作风格指南（2026修订版）

> 本指南经过2026年5月自我评审修订，新增场景密度强制标准和具体写作要求。

---

## 一、写作质量强制标准

### 1.1 场景密度标准（必须达到）

| 检查项 | 强制标准 | 如何判断 |
|--------|---------|---------|
| 感官细节 | 每500字至少1处 | 数全文有多少"闻到""听到""摸到"的描述 |
| 对话场景 | 每1000字至少1处 | 数全文有多少真实的对话交锋 |
| 内心独白 | 每150字封顶 | 连续内心独白段落超过150字=不合格 |

**什么是感官细节**：
- 气味：机油味、潮湿的空气、烧焦的味道
- 声音：风扇的嗡嗡声、远处的喊声、沉默
- 温度：微凉的潮湿、冰冷的触感、闷热
- 触感：粗糙的墙壁、滑腻的汗水、柔软的棉袄
- 视觉：光线、颜色、动态、静止

### 1.2 禁止出现的写法

| 禁止项 | 错误示例 | 正确替代 |
|--------|---------|---------|
| 情绪词替代行为 | "她非常难过" | "她把手机屏幕扣过去，扣在餐桌上，这个动作她做了好几次" |
| 直接说出心理 | "那一刻他突然意识到……" | 用场景和行为暗示内心 |
| 议论替代叙事 | "摩托车和汽车不一样……" | 停在"说"，进入场景 |
| 大词空泛 | "广州天河区的写字楼" | "南山科技园2栋12层1208" |
| 流量词汇 | "扎心""破防""泪崩" | 一旦出现，质感报废 |
| 内心独白泛滥 | 连续超过150字的"我想""我觉得" | 改为场景或对话 |

## 二、开篇技法

### 2.1 禁止开头（一律不得出现）

```
❌ 「她叫XXX，今年XX岁，是一名普通的XXX。」
❌ 「最近，一件事深深地触动了我的心弦……」
❌ 「人生就像一场旅行，不在乎目的地，在乎的是沿途的风景……」
❌ 「这件事要从XX年前说起……」
❌ 「XX的故事，要从……说起」
```

### 2.2 正确开头（直接进入场景）

**公式**：时间 + 地点 + 人物动作 → 让读者"看见"场景

**经典案例**：
```
「她在赤水河边站了很久。三月的风从河谷里吹上来，带着一种微凉的潮湿。
远处有人在喊她的名字，她没有应答，只是把手里那件红色的棉袄叠了叠，
放在了岸边的石头上。」
```

**《外卖骑手》开头**：
```
「那天系统给他派了三单。三单，三条路线，互相矛盾。
他看了一眼手机屏幕，外面的雨已经下大了。」
```

**开篇三要素**：
- 具体时刻：不是"有一天"，而是"那天下午三点""周五下午两点"
- 具体地点：不是"某地"，而是"赤水河边""公司门口"
- 具体动作：不是"她很难过"，而是"她把棉袄叠了叠，放在石头上"

## 三、对话技法

### 3.1 好对话的三特征

1. **口语化**：有语气词、停顿、不完整的句子
2. **有性格**：读者的"耳朵"能分辨说话的人
3. **不承担叙事功能**：对话展现人物，不推进情节

### 3.2 对话正确示范

```
正确：「'你到底想要什么？'
他没回答，只是把自己的外套拿起来，走到门口，又停了一下。
'我只是觉得，'他说，'算了，没什么。'」

正确：「'你被裁了。'经理说完，等了一秒，像是在等我哭。
我没哭。他把桌上的纸巾盒往我这边推了推。
我说：'不用。'」
```

### 3.3 对话禁止示范

```
❌ 「她说：'你怎么了？'我说：'我被裁员了，很难过。她安慰我说一切都会好起来的。'」
→ 这是对话承担叙事功能，人物没有性格，读者听不见声音

❌ 「经理说：'公司决定裁员，你被列入名单了。'我问：'为什么？'经理说：'业绩不好。'」
→ 这是问答式对话，没有停顿，没有动作，没有性格
```

### 3.4 对话后的处理

- 对话后不需要总是加"她说"，可以用动作交代
- 对话与叙述交替，避免大段连续对话
- 好的对话让读者"听见"声音

## 四、内心独白技法

### 4.1 核心原则

```
内心独白只能在充分观察的基础上偶尔使用。
连续超过150字的内心独白段落 = 不合格。
```

### 4.2 替代方案

| 内心独白写法 | 场景替代 |
|-------------|---------|
| "他很紧张" | "他的手指在裤腿上蹭了两下，又抬起来，把水杯盖子拧开，拧上，又拧开" |
| "他突然明白了" | 让他说："我想我明白了" |
| "她不知道该说什么" | "她把头转过去，看向窗外" |
| "他觉得很累" | "他坐在椅子上，背靠在墙上，眼睛闭了三秒钟" |

### 4.3 内心独白的正确用法

**仅在以下情况使用内心独白**：
- 充分观察后的心理活动（已经写了大量场景和对话）
- 读者已经通过场景理解了人物处境
- 心理活动是对前面场景的"回应"，不是"代替"

## 五、结尾技法

### 5.1 禁止结尾

```
❌ 「这件事让我们明白，人生没有白走的路。」
❌ 「只要心中有梦想，就一定能成功。」
❌ 「生命是短暂的，我们要珍惜当下。」
❌ 「那一刻，人们终于明白了生命的意义。」
```

### 5.2 正确结尾

**停在故事的时间点上，让现实说话**：

```
正确：「她现在住在西安郊区的一栋老楼里。
偶尔她会坐两个小时的公交车，去当年的那条河边坐一坐。
河水还在流。她把鞋子脱了，脚放进水里，什么都不想。」
```

```
正确：「他站在那儿没动，直到服务员来问他要不要续水，
他才意识到时间已经过了很久。」
```

**正确结尾三特征**：
- 不升华，不总结
- 让现实本身说话
- 读者走出去还在想

## 六、标题技法

### 6.1 标题三禁止

| 禁止项 | 错误示例 |
|--------|---------|
| 结论式 | 《这是我做过最正确的决定》 |
| 感叹号 | 《太震撼了！》 |
| 流量词 | 《这个故事让我泪崩了》 |

### 6.2 标题模板

| 类型 | 公式 | 示例 |
|------|------|------|
| 意象型 | [意象] + [人物/处境] | 《城堡里的马原》《红衣女人》 |
| 问题型 | 让人想知道答案的问句 | 《那个人为什么不跑？》 |
| 动词型 | [动词] + [人物] | 《困在系统里的人》《成为张桂梅》 |
| 地点型 | [地点] + [人物状态] | 《长江大桥上的守桥人》 |
| 隐喻型 | 文学意象含第二层含义 | 《平原上的娜拉》 |

### 6.3 标题检查

**标题写完后问自己**：读者看完标题，是否知道文章要说什么？
- 知道 = 标题泄露了，不合格
- 不知道但有想知道的冲动 = 合格

---

## 七、节奏控制

### 7.1 文章必须有"山"

**均匀推进 = 流水账，不是故事**

```
错误结构：
开篇 → 城市1 → 城市2 → 城市3 → 想通了 → 结尾
（每个部分权重相等，没有高潮）

正确结构：
开篇(直接进入困境) → 中段(场景推进，每1500字一个小高潮) → 高潮(放慢，增加细节) → 收尾
```

### 7.2 高潮点的判断

**高潮点三特征**：
1. 全篇情感密度最高处
2. 放慢节奏，增加细节
3. 读者应该感到"呼吸变重"

**如果找不到高潮点 = 没有山，不合格**

---

## 八、写作自检清单

写完每篇文章后，逐项检查：

| 检查项 | 合格标准 | 不合格则 |
|--------|---------|---------|
| 困境测试 | 能说清"主人公选了X，代价是Y" | 重选困境 |
| 开篇 | 直接进入场景，无自我介绍 | 重写开篇 |
| 感官细节 | 每500字至少1处 | 补充细节 |
| 对话场景 | 每1000字至少1处 | 补充对话 |
| 内心独白 | 纯内心段落不超过150字 | 改为场景 |
| 主人公缺陷 | 有让读者不认同的时刻 | 增加不完美 |
| 高潮点 | 有全篇情感密度最高处 | 重写高潮 |
| 结尾 | 不升华，不总结 | 重写结尾 |
| 标题 | 是意象/问题，不是结论 | 重写标题 |
| 流量词 | 无"扎心""破防""泪崩" | 删除替换 |

## 九、经典段落样本

### 开篇样本

> 她在赤水河边站了很久。三月的风从河谷里吹上来，带着一种微凉的潮湿。远处有人在喊她的名字，她没有应答，只是把手里那件红色的棉袄叠了叠，放在了岸边的石头上。

→ 特征：无自我介绍，直接进入时刻，有感官细节

### 对话样本

> "你到底想要什么？"
> 他没回答，只是把自己的外套拿起来，走到门口，又停了一下。
> "我只是觉得，"他说，"算了，没什么。"

→ 特征：不完整的句子、停顿、动作交代

### 结尾样本

> 她现在住在西安郊区的一栋老楼里。偶尔她会坐两个小时的公交车，去当年的那条河边坐一坐。河水还在流。她把鞋子脱了，脚放进水里，什么都不想。

→ 特征：不升华，停在当下，让现实说话
