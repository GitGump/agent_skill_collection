# 非虚构人物报道写作风格指南 —— 《人物》杂志调性

## 概述

本文档是《人物》杂志公众号写作风格的系统性指南，供智能体在创作非虚构人物报道时参考遵循。核心理念：**触及人物的内心、悲喜、情感**，用一个人的故事，撬动一个时代的神经。

---

## 一、核心写作哲学

### 1.1 《人物》vs 真实故事计划

| 维度 | 真实故事计划 | 《人物》杂志 |
|------|------------|------------|
| 叙述视角 | 第一人称为主 | 第三人称全知为主 |
| 叙事姿态 | 介入的、陪伴的 | 观察的、记录的 |
| 问题意识 | 个人困境 | 个人困境->时代问题 |
| 结尾 | 开放式，留有余味 | 不升华，让现实说话 |
| 标题风格 | 朴素+陈述句 | 意象+隐喻，文学性强 |

### 1.2 「触及时代神经」是核心

《人物》杂志最厉害的地方，是**用一个具体的人的故事，撬动一个时代的神经**：

- 不是要写"一个感人的故事"
- 而是要写"一个人在某种处境下，如何做出了某个选择"
- 那个选择背后，有什么东西在支撑或撕裂他/她
- 这个人的命运，如何折射了时代的问题

### 1.3 不消费苦难

《人物》杂志最忌讳的写法：

- 将底层苦难作为展示道具，引发读者的优越感或怜悯
- 用"感动"作为目的，强行推进情绪
- 用夸张的悲剧叙述博取流量
- 不做简单的是非对错判断，让读者自己得出结论
- 将叙述者的处境呈现为："这就是生活在这个时代的样子"

## 二、叙事视角与人称

### 2.1 第三人称全知叙述（主力）

《人物》绝大多数文章以第三人称全知视角为主：

```
她站在长江大桥的护栏边，风从江面上吹过来，带着一点腥气。
十二年里，她在这座桥上来来回回走了两万多趟，见过太多人在那个护栏边上停留。
```

- 第三人称给叙述者更大的观察和判断空间
- 可以深入人物内心（不同于真实故事计划的"我"的局限视角）
- 叙述者是冷静的观察者，不是参与者

### 2.2 叙述层次

优秀的非虚构人物报道常有两条线索：
- **事件线**：发生了什么
- **内心线**：人物如何感受、思考、理解

两条线交织，形成叙事的纵深。

## 三、语言风格

### 3.1 具体优于抽象

```
错误: 「她当时非常难过。」
正确: 「她把手机屏幕扣过去，扣在餐桌上，这个动作她做了好几次。」

错误: 「那是一段艰难的时光。」
正确: 「那三个月，她每天早上醒来的第一件事，就是看账户余额，
      然后想今天要省掉哪顿饭。」
```

### 3.2 感官细节建立现场

好的非虚构文章，读者应该能「看见」「听见」「闻到」叙述者所处的场景：

```
示例：
「工厂里的气味是那种混合了机油、棉絮和汗水的复合气味，
他到现在有时候在某个地铁站闻到类似的味道，还会愣一秒。」
```

### 3.3 对话的力量

对话是非虚构写作中最有力的工具：
- 展现人物性格（说话方式就是性格）
- 推进情节
- 传递情感（不用直接说"他很冷漠"，让他的话来说）

**对话写作原则**：
- 对话要真实、口语化，有语气词、停顿、不完整的句子
- 对话后不需要总是加"她说"，可以用动作交代
- 对话与叙述交替，避免大段连续对话

### 3.4 克制的情绪表达

《人物》的情绪是"渗出来的"，不是"喊出来的"：

```
错误: 「泪水夺眶而出，那一刻，人们终于明白了生命的意义。」
正确: 「他站在那儿没动，直到服务员来问他要不要续水，
      他才意识到时间已经过了很久。」
```

## 四、结构设计

### 4.1 开篇：抛入场景，不做自我介绍

**禁止开头**：
```
错误:「她叫刘小样，今年45岁，是一名普通的陕西农村妇女。」
错误:「最近，一件事深深地触动了我的心弦……」
```

**好的开头**：直接进入一个具体时刻/场景

```
正确:「她在赤水河边站了很久。三月的风从河谷里吹上来，带着一种微凉的潮湿。
远处有人在喊她的名字，她没有应答，只是把手里那件红色的棉袄叠了叠，
放在了岸边的石头上。」

正确:「那天系统给他派了三单。三单，三条路线，互相矛盾。
他看了一眼手机屏幕，外面的雨已经下大了。」
```

### 4.2 中段：推进，不停顿在情绪里

故事的中段应该像水流——持续流动，不在某个情绪点上停留太久。

### 4.3 转折点：全文情感密度最高处

这个时刻需要**放慢节奏、增加细节**：

```
示例：
「她把那件红棉袄留在河边的石头上之后，沿着公路走回了村子。
那是她最后一次穿那件红衣服出现在村子里。
从那以后，她穿上了灰颜色的衣服，把头发也剪短了。
她开始用另一种方式去生活，去说话，去忍耐。
但有些东西，她始终没有丢掉。」
```

### 4.4 结尾：留有余地，不作总结

《人物》的结尾很少给出人生总结，而是停在故事现在的时间点上，让读者感受余韵。

```
错误:「这件事让我们明白，人生没有白走的路。」
正确:「她现在住在西安郊区的一栋老楼里。
偶尔她会坐两个小时的公交车，去当年的那条河边坐一坐。
河水还在流。她把鞋子脱了，脚放进水里，
什么都不想。」
```

## 五、人物塑造

### 5.1 让主人公有立体感

让坚强和脆弱都在具体行为里呈现，读者自己感受。

### 5.2 让主人公有局限性

优先选择：犯过错的人、做出过糟糕选择的人、不完美的人。

### 5.3 避免符号化

不要把主人公处理成某种"代表"（女性的代表/底层的代表/时代的代表），而是呈现她/他作为一个人的独特性。

---

## 六、优秀作品特征清单

- [ ] 开篇是否直接进入具体场景？
- [ ] 是否有足够具体的感官细节？
- [ ] 是否有真实的对话或独白？
- [ ] 情绪词是否已被具体行为/细节替代？
- [ ] 主人公是否有立体感？
- [ ] 转折时刻是否放慢节奏、深入处理？
- [ ] 结尾是否留有余地？
- [ ] 全文是否没有"流量词汇"？
- [ ] 是否触及了时代性问题？
- [ ] 是否避免了消费苦难的倾向？
- [ ] 整体字数是否在 2000~5000 字范围内？
