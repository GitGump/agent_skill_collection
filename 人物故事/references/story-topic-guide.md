# 选题策略、采访框架与标题指南 —— 人物故事

## 一、选题策略

### 1.1 什么样的故事值得写？

**核心判断标准（人物故事四问法）**：

1. **有人吗？** 这个故事里有一个真实的、有血有肉的人
2. **有困境吗？** 这个人面对了什么无法轻易解决的困境
3. **有挣扎吗？** 这个人在其中做了选择、付出了代价
4. **有时代性吗？** 这个人的故事能否触及时代的神经？

满足前三点，就有了基本的非虚构叙事土壤；满足第四点，就有了《人物》级别的传播力。

---

### 1.2 选题方向矩阵

#### 争议人物类
| 故事类型 | 关键叙事点 | 代表作品 |
|---------|-----------|---------|
| 以爱为名的控制 | 家庭关系中的权力与边界 | 《城堡里的马原》 |
| 公众人物的另一面 | 顶流光环下的真实困境 | 《幸存者李佳琦》 |
| 知识分子的困境 | 理想主义与现实的错位 | 《神童到中年》 |

#### 命运转折者类
| 故事类型 | 关键叙事点 |
|---------|-----------|
| 突然走红的人 | 流量时代的命运翻转 |
| 被时代遗忘的人 | 改革浪潮中的失落者 |
| 选择的代价 | 重大决定之后的生活 |

#### 边缘/普通人群类
| 故事类型 | 关键叙事点 | 代表作品 |
|---------|-----------|---------|
| 被系统困住的人 | 算法时代的人力困境 | 《外卖骑手，困在系统里》 |
| 乡村留守者 | 城乡撕裂中的个体 | 《平原上的娜拉》 |
| 漂泊者 | 城市里没有归属感的人 | 城漂系列 |

#### 社会热点类
| 故事类型 | 关键叙事点 | 风险提示 |
|---------|-----------|---------|
| 灾难/悲剧亲历者 | MU5735等重大事件 | 伦理边界，需极度克制 |
| 犯罪/法律边缘 | 灰色地带的人物 | 避免猎奇，需有社会洞察 |
| 性别议题 | 女性/性少数群体 | 避免符号化，需有复杂性 |

---

### 1.3 选题禁忌

- 单纯的成功故事（若没有困境与代价，就缺乏真实感）
- 极端猎奇故事（消费人物的不幸）
- 明显的正能量框架（一开始就指向"坚强""奋斗"）
- 纯粹的受害者叙事（只呈现被动受苦，没有主体性）

## 二、采访框架（当用户提供素材时）

### 核心问题

**基础信息**：
- 主人公年龄大概多少？
- 在哪里发生的？
- 故事发生在什么时间范围内？

**核心叙事信息**：
- 故事"起点事件"是什么？
- 最重要/最难忘的一个场景或时刻是什么？
- 主人公面对的最核心困境是什么？
- 主人公做了哪些选择？付出了什么代价？
- 故事现在在哪里？（已结束/还在进行中）

**细节挖掘**：
- 有没有一段印象深刻的对话？
- 有没有一个很具体的物品/场景？
- 他/她有什么习惯性的动作或说话方式？

## 三、标题指南

### 3.1 《人物》杂志标题特征

| 特征 | 说明 |
|-----|------|
| 朴素 | 不用感叹号，不用煽情词 |
| 意象感 | 常用一个核心意象（城堡、红衣、系统） |
| 有隐喻 | 标题本身含有第二层含义 |
| 不解释 | 不在标题里泄露情绪或立场 |
| 文学性 | 像一个文学句子，有韵律感 |

### 3.2 代表作品标题分析

```
《平原上的娜拉》
—— "平原"是地理意象，"娜拉"是隐喻（易卜生《玩偶之家》）
—— 指向：中国农村女性的觉醒与困境

《外卖骑手，困在系统里》
—— "困在"是动词，制造张力
—— "系统"是抽象词，指向时代命题

《城堡里的马原》
—— "城堡"是核心意象：既是庇护，也是牢笼
—— 悬念：城堡里发生了什么？

《成为张桂梅》
—— "成为"是过程，是动词，是动态的

《幸存者李佳琦》
—— "幸存者"一词暗含艰辛（幸存=经历灾难后活下来）
—— 重新定义"成功"的光环
```

### 3.3 标题生成模板（人物故事风格）

**类型 A：意象+人物型**
```
结构：[核心意象] + [人物/处境]
示例：
- 《红衣女人》
- 《城堡里的作家》
- 《河边的女人》
- 《系统里的人》
```

**类型 B：动词+人物型**
```
结构：[动词] + [人物/职业]
示例：
- 《困在算法里的人》
- 《成为张桂梅》
- 《消失的骑手》
```

**类型 C：地点+人物型**
```
结构：[地点/场景] + [人的状态]
示例：
- 《长江大桥上的守桥人》
- 《ICU里的春天》
- 《凌晨四点的工厂》
```

**类型 D：群像型**
```
结构：[事件/状态] + 的 + [群体名]
示例：
- 《MU5735航班上的人们》
- 《那些被困在系统里的年轻人》
```

**类型 E：隐喻型**
```
结构：用文学意象暗示核心主题
示例：
- 《城堡里的马原》（城堡=庇护/牢笼/极权）
- 《平原上的娜拉》（娜拉=觉醒/出走/困境）
```

### 3.4 标题禁忌

- 感叹号：「太震撼了！」「这个故事让我哭了！」
- 网络流量词：「扎心」「破防」「泪崩」「感动了无数人」
- 过度悬念：「没想到……」「结局出乎意料」
- 定性评价：「最感人」「最伟大」「最坚强」
- 空洞大词：「生命的意义」「活着的价值」
