# 选题策略、采访框架与标题指南（2026修订版）

> 本指南经过2026年5月自我评审修订，新增"困境优先"选题标准。

---

## 一、选题策略

### 1.1 困境测试（最重要）

**核心判断标准**：每个选题必须通过困境测试。

**困境测试四问**：
1. **有人吗？** 有一个真实的、有血有肉的人
2. **有困境吗？** 困境是两难的，两个选项都有代价，无法轻易解决
3. **有代价吗？** 选择之后，有些东西永远失去了
4. **有时代性吗？** 一个人的故事能撬动时代的神经

**困境测试失败 = 不值得写**

通过测试：能用一句话说清"主人公选择了X，代价是Y"
未通过测试：说不清楚，只是流水账

### 1.2 好困境的特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 两难选择 | 两条路都有代价 | 出走 vs 留守，救 vs 不救 |
| 代价具体 | 可感、可量化 | 见孩子从每周变每年，母亲关上门 |
| 无法回头 | 选了就回不去了 | 有些东西永远失去 |
| 让读者纠结 | 不只是主人公在纠结 | 读者也犹豫"我会怎么选" |

### 1.3 选题升级案例

| 弱选题（无困境） | 问题 | 强选题（有困境） |
|-----------------|------|-----------------|
| 被裁员后去旅行散心 | 旅行没有代价，只是散心 | 被裁员后妻子提出离婚，他选择不争抚养权去旅行 |
| 程序员回老家创业 | 代价不够具体 | 回老家后，母亲说"你是家里唯一的大学生"，孩子留在城市 |
| 普通人的挣扎 | 没有两难选择 | 他决定不救父亲，因为救下来也是植物人 |
| 女性的觉醒 | 太抽象 | 她想出走，但孩子才三岁 |

### 1.4 选题方向矩阵

#### 争议人物类
| 故事类型 | 核心困境 | 代表作品 |
|---------|---------|---------|
| 以爱为名的控制 | 父亲的信念 vs 儿子的命 | 《城堡里的马原》 |
| 公众人物的另一面 | 成功 vs 焦虑 | 《幸存者李佳琦》 |
| 知识分子的困境 | 理想 vs 家庭 | 多篇知识分子特写 |

#### 命运转折者类
| 故事类型 | 核心困境 |
|---------|---------|
| 突然走红的人 | 流量 vs 真实生活 |
| 被时代遗忘的人 | 当年 vs 现在 |
| 选择的代价 | 选择了X，失去了Y |

#### 边缘/普通人群类
| 故事类型 | 核心困境 | 代表作品 |
|---------|---------|---------|
| 被系统困住的人 | 效率 vs 尊严 | 《外卖骑手，困在系统里》 |
| 乡村留守者 | 出走 vs 留守 | 《平原上的娜拉》 |
| 城市漂泊者 | 留下 vs 离开 | 城漂系列 |

### 1.5 选题禁忌

| 禁忌类型 | 问题 | 替代方向 |
|---------|------|---------|
| 单纯的成功故事 | 没有困境与代价 | 加上选择的代价 |
| 极端猎奇故事 | 消费人物的不幸 | 聚焦困境而非猎奇 |
| 正能量框架 | 指向"坚强""奋斗" | 让困境本身说话 |
| 纯受害者叙事 | 没有主体性 | 给主人公选择的时刻 |

## 二、采访框架

### 2.1 核心问题

**困境挖掘**：
- 主人公面临的最艰难的选择是什么？
- 两个选项各是什么？各有什么代价？
- 选择之后，什么东西永远失去了？
- 如果重来一次，还会这样选吗？

**场景挖掘**：
- 最难忘的一个时刻是什么？具体在哪里？发生了什么？
- 有没有一段印象深刻的对话？
- 有没有一个具体的物品或场景？

**细节挖掘**：
- 主人公有什么习惯性动作？
- 说话的方式是什么？有什么口头禅？
- 有没有一个反差很大的细节？

### 2.2 多元视角

采访至少3类人：
- 本人
- 家人/亲密关系
- 旁观者/对手

## 三、标题指南

### 3.1 标题三禁止

| 禁止项 | 错误示例 | 说明 |
|--------|---------|------|
| 结论式 | 《这是我做过最正确的决定》 | 泄露答案 |
| 感叹号 | 《太震撼了！》 | 流量嫌疑 |
| 流量词 | 《这个故事让我泪崩了》 | 质感报废 |

### 3.2 标题模板

**类型A：意象+人物**
```
结构：[核心意象] + [人物/处境]
示例：
- 《城堡里的马原》
- 《红衣女人》
- 《河边的她》
```

**类型B：动词+人物**
```
结构：[动词] + [人物/职业]
示例：
- 《困在系统里的人》
- 《成为张桂梅》
- 《消失的骑手》
```

**类型C：地点+人物**
```
结构：[地点/场景] + [人的状态]
示例：
- 《长江大桥上的守桥人》
- 《ICU里的春天》
- 《凌晨四点的工厂》
```

**类型D：隐喻型**
```
结构：用文学意象暗示核心主题
示例：
- 《平原上的娜拉》（平原=地理，娜拉=觉醒隐喻）
- 《城堡里的马原》（城堡=庇护/牢笼/极权）
```

### 3.3 标题检查

**问自己**：读者看完标题，是否知道文章要说什么？
- 知道 = 标题泄露了，不合格
- 不知道但有想知道的冲动 = 合格

---

## 四、标题自检表

| 检查项 | 合格标准 |
|--------|---------|
| 意象感 | 有可视化的意象或场景 |
| 隐喻性 | 标题含有第二层含义 |
| 悬念感 | 不泄露答案 |
| 无感叹号 | 没有"！" |
| 无流量词 | 没有"扎心""破防""泪崩" |
