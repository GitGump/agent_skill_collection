# 公众号排版优化参考 —— 人物故事

## 一、整体基调

- **克制、沉稳、留白充足**
- 不用醒目配色，主色调为深灰/黑色
- 文字密度不过高，段落间有呼吸感
- 大行间距，沉浸阅读体验

## 二、排版规则

### 2.1 结构设计
- 不使用大量小标题分割（非干货文，避免碎片化）
- 若需分段，用**数字序号**（一、二、三）或空行分隔
- 金句用引用样式（`>` 块）突出
- **不使用任何分隔线**（`---` / `———`）

### 2.2 段落处理
- 段落首行不缩进
- 段落之间留有呼吸感（1~2行的空行）
- 每个段落不宜过长，避免阅读疲劳
- 每个段落也不宜过短（低于3行），否则节奏破碎

### 2.3 对话处理
- 对话文字适当缩进或使用引用样式
- 保持对话口语化，不做过多修饰

### 2.4 重点处理
- 重点信息可加粗，但不滥用
- 每月用2~3次加粗，每次不超过2个词

## 三、口述引言格式

```
> [主人公姓名]，年龄，籍贯。职业/身份简述。
> 一个具体的处境或困境描述。
> 以下是他的讲述。
```

## 四、隐私声明格式

文章末尾加一行：

```
> （出于隐私保护，文中人物为化名）
```

## 五、互动引导语

文章末尾，隐私声明之前：

```
如果你也有一个想被记录的故事，欢迎留言。
```

| 正文字色 | #2d2d2d |

| 引言色 | #555555 |
| 背景色 | #ffffff |

| 段间距 | 20px |
