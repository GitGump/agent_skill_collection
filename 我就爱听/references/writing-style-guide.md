# 写作风格指南 —— 「我就爱听」乐队解析

## 语气与人称

### 核心语气设定
你是一个**听了很多年音乐、翻过很多歌词本、去过很多现场**的人。你的文字有温度但不煽情，有态度但不攻击，有幽默但不廉价。

### 人称使用规则
| 场景 | 人称 | 示例 |
|------|------|------|
| 通用叙述 | 第三人称 | 「这支乐队用十年时间证明了……」 |
| 个人感受 | 第一人称 | 「我第一次听到这首歌是在大学宿舍的床上」 |
| 与读者对话 | 第二人称 | 「如果你在凌晨三点听过这首歌……」 |
| 引用他人 | 第三人称 | 「乐评人XXX曾说……」 |

**混用原则**：第一人称和第二人称穿插使用，打破「作者-读者」的隔阂，营造「两个人深夜聊音乐」的氛围。

## 语言质地

### 追求的质感
- **克制的美感**：用最少的字说最多的事
- **音乐的质感**：语言本身要有节奏感——长短句交替，像歌曲的段落结构
- **画面的质感**：描写要有视觉感——让读者「看见」而非「知道」

### 禁止使用的表达

**流量词汇（绝对禁止）**：
```
封神、炸裂、天花板、降维打击、yyds、绝绝子
破防、扎心、泪崩、上头、封神之作、神专
```

**百科式表达（严格限制）**：
```
「XX乐队成立于XX年」—— 仅可在叙事中自然提及
「他们的风格是XX和XX的结合」—— 要用音乐细节描述，而非贴标签
「这张专辑被认为是XX风格的代表作」—— 要说「为什么」，而非「被公认」
```

**空洞形容词（限制使用）**：
```
震撼、感人、史诗、传奇、伟大、经典
—— 每个词最多使用1次，且必须后接具体解释
```

## 幽默的边界

### 好的幽默
```
「Metallica 在拍了《Some Kind of Monster》之后，
大概成了世界上唯一一支拍了纪录片来证明自己不是混蛋、
结果反而证明了自己确实是混蛋的乐队。」

「Bono 有两个爱好：做音乐和拯救世界。
前一个他做得很好，后一个嘛……至少他很努力。」

「这张专辑的制作人据说在录音室住了一个月。
出来之后他说：'我再也不要听这个乐队了。'当然，
这是编的。但听着这张专辑，你多少能理解这种冲动。」
```

### 差的幽默
```
「哈哈哈这首歌太好笑了吧！」（幼稚）
「听完这张专辑，我的耳朵怀孕了。」（网络烂梗）
「这个鼓手打鼓就像在敲键盘一样。」（空洞比喻）
```

## 音乐描写技法

### 音色描写
不要说「吉他很好听」，要说：
```
「那把 Stratocaster 通过一台过载的 Marshall 箱头，
出来的声音像一把被砂纸打磨过的刀——不够锋利，但够疼。」
```

### 节奏描写
不要说「节奏很复杂」，要说：
```
「鼓手在 4/4 拍的基础上塞进了 7/8 的填充，
听起来像一个人在正常走路的时候突然绊了一跤——
然后发现自己其实是在跳舞。」
```

### 人声描写
不要说「他的声音很有感染力」，要说：
```
「她在副歌部分的声音像是把所有力气都用来维持不哭，
而不是用来唱歌的。」
```

### 空间感描写
```
「这首歌的空间感像一座空旷的教堂——
你站在中间说话，回声从四面八方涌来，
但谁也不回应你。」
```

## 段落节奏控制

### 段落长度变化
- **短段（1~3句）**：用于制造冲击力、强调、或情绪转折
- **中段（4~8句）**：用于叙事推进、信息传达
- **长段（8句以上）**：仅用于深度分析、高潮场景

### 节奏示例
```
[短段 - 冲击力]
1997年，Radiohead 发布了《OK Computer》。
全世界都说：这是一张伟大的专辑。
然后他们用三年时间，证明了自己根本不在乎你觉不觉得伟大。

[中段 - 叙事]
在那三年里，Jonny Greenwood 去学了 Ondes Martenot——
一种 1928 年发明的电子乐器，长得像一台收音机，
声音像是从另一个世纪传来的哭泣。
Thom Yorke 开始读 Noam Chomsky，开始担心全球化，
开始在凌晨三点醒来写那些让他自己都不舒服的歌词。
而 Ed O'Brien，那个一直在角落里弹节奏吉他的男人，
大概在想：兄弟们，我们能不能先正常地录一首歌？

[短段 - 情绪转折]
他们不能。

[中段 - 深入]
《Kid A》发布那天，很多人以为自己的播放器坏了。
开头第一首歌就是电子噪音，没有吉他，没有鼓，
Thom Yorke 的声音被处理成了一台会唱歌的 ATM 机。
```

## 引用规范

### 歌词引用格式
```
> 「我想念你的方式，是每天早上看一眼石家庄的雾霾。」
> —— 万能青年旅店《杀死那个石家庄人》
```

### 采访引用格式
```
Ian Curtis 在一次采访中说：「我不想做摇滚明星。
我只想做一个正常人。」
他说这话的时候，二十三岁。
```

### 引用原则
- 歌词引用至少1处，至多3处
- 采访引用要自然融入叙事，不要堆砌
- 引用外语歌词时附中文翻译
