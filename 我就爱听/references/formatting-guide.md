# 排版指南 —— 「我就爱听」乐队解析

## 整体基调

音乐杂志风格：有设计感，但不花哨。像一个好品味的人做的个人博客。

---

## 标题层级

### 文章标题
- 使用一级标题 `#`
- 标题风格：意象/问题/场景，不是结论
- 不使用感叹号

### 正文章节标题
- 使用二级标题 `##`
- 章节标题要有趣味性，不是简单的「一、乐队简介」
- 好的标题 vs 差的标题：

| 差的章节标题 | 好的章节标题 |
|------------|------------|
| 乐队简介 | 石家庄的地下室 |
| 成员介绍 | 那个不想当主唱的吉他手 |
| 专辑分析 | 一张杀死所有人的专辑 |
| 推荐歌单 | 如果你只有十分钟 |
| 结尾 | 今天晚上，戴上耳机 |

---

## 排版规则

### 歌词引用
```markdown
> 「我想念你的方式，是每天早上看一眼石家庄的雾霾。」
> ——《杀死那个石家庄人》
```

### 采访引用
```markdown
Ian Curtis 在一次采访中说：「我不想做摇滚明星。我只想做一个正常人。」
```
（不使用引用块，直接融入段落）

### 歌曲名与专辑名
- 歌曲名用中文书名号：「Creep」「杀死那个石家庄人」
- 专辑名用英文书名号：《OK Computer》《冀西南林路行》
- 中英混用时统一风格

### 章节分隔
- **不使用任何分隔线**（`---` / `———`）
- 用空行分隔段落
- 用二级标题标记章节

### 推荐歌单排版
```markdown
## 推荐歌单

🎧 **入门级**——如果你从没听过他们
- 「Creep」——你一定在某处听过这首歌的旋律，但带着耳机重新听一次，你会发现它比你以为的更悲伤
- 「Karma Police」——这首会让你觉得，这个世界确实需要一些正义

🔄 **进阶级**——如果你听过几首，想深入了解
- 「Paranoid Android」——6分半的史诗，包含三个完全不同的段落，像一部浓缩的电影

🔥 **死忠级**——如果你已经爱了，这些是必听
- 「The National Anthem」——混乱、嘈杂、不和谐，但这正是它的伟大之处

💎 **私藏级**——这些是我个人的私藏
- 「Life in a Glasshouse」——爵士、混乱、美得让人不安。这张专辑最后的一首歌，也是最被低估的一首。
```

## 段落格式

### 段落长度
- 普通段落：3~8 句
- 冲击性短段：1~3 句（独立成段，前后各空一行）
- 长段分析：不超过 10 句（超过则拆分）

### 特殊排版效果

**单句成段**（制造冲击力和节奏变化）：
```
他们不能。

这三分钟，改变了摇滚乐。

那是一把 1962 年的 Fender Stratocaster。
他花了 200 美元买的。
现在它价值 100 万美元。
但它真正的价值不是价格——是它在那个录音室里发出的声音。
```

**对话式插入**（增强亲切感）：
```
你可能在某个深夜刷到过这首歌。
可能在某个朋友的车上听过。
可能在一个你没注意到的广告背景音乐里听到过。
但你可能从来没有认真听过它。
```

## 结尾格式

### 推荐互动语
```
如果你也是这支乐队的听众，
在评论区告诉我你最喜欢他们的哪首歌。
或者哪张专辑。
或者哪个现场。
我很好奇。
```

### 禁止的结尾
- ❌ 「如果你喜欢这篇文章，请点赞关注！」
- ❌ 「XX乐队永远是最伟大的乐队！」
- ❌ 「音乐就是生命，让我们一起感受音乐的力量吧！」
- ✅ 一句歌词、一个画面、一个反问，然后安静地结束
